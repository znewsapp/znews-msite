每次看球賽，都好奇隊醫的小箱箱裏裝了些什麼

足球比賽現場隊醫的藥箱會準備哪些東西？

本答案較長，希望在爲題主解惑之外，也能爲草根隊醫提供幫助，減少業餘足球愛好者的傷痛。因此下文會分爲兩部分討論，第一部分介紹專業隊醫裝備（譯自國際足聯官方指南），第二部分是給草根隊醫的建議，如有疏漏錯誤，懇請指正。

專業隊醫包裏有什麼？

隊醫到底需要做什麼？專業隊醫（team physician）的職責比我們想象的要複雜：

  1. 傷害管理
  2. 疾病管理
  3. 傷病預防
  4. 急救
  5. 適應性問題（時差、炎熱、潮溼和海拔）
  6. 反興奮劑
  7. 優化表現
  8. 諮詢

FIFA 一直對足球中的健康問題保持關注。1994 年，在布拉特的主持下，國際足聯成立了醫學評估和研究中心（F-MAC），負責發展足球傷害預防，以及關懷標準的改進。該機構發行了兩版 F-MAC 足球醫學指南，指南中明確寫出了隊醫藥箱（主藥箱 + 急救包）的配置， 均由藥品和設備兩部分組成。

急救包：

▍藥品

  • 腎上腺素注射液（1:1000）
  • 阿托品注射液（1 mg）
  • 氫化可的鬆注射液
  • 氯苯那敏注射液（又稱撲爾敏，10 mg）

▍設備

  • 面罩
  • 口腔導氣管
  • 頸託
  • 冷卻噴霧
  • 快速止血棉
  • 繃帶（有膠 / 無膠）
  • 手套
  • 三角繃帶
  • 棉花、紗布、眼墊
  • 縫合用膠
  • 剪刀
  • 酒精棉片

主藥箱：

▍藥品

【鎮定類】

1.非類固醇消炎藥（如雙氯芬酸、布洛芬）

2.曲馬多注射液

3.撲熱息痛、可待因酮

4.肌鬆藥

5.注射用糖皮質激素（如曲安西龍）

6.COX-2 抑制劑

【腸胃類】

7.鎮吐藥（口服、注射）

8.止瀉藥（如洛哌丁胺）

9.大便鬆軟劑

10.口服補液鹽

11.解酸劑、奧美拉挫、雷尼替丁

12.口嚼維 C

【呼吸】

13.哮喘用吸入器（如舒喘寧，苄氯丙酰胺噴霧）

14.止咳化痰劑

【傷口和皮膚清理】

15.雙氧水、沙威隆、滴露、異丙醇

【耳鼻喉】

16.耳蠟鬆軟劑

17.眼、耳用抗生素

18.抗組織胺眼藥水

19.抗組織胺藥（如非索非那定、氯雷他定）

20.鼻炎噴霧（解充血藥如賽洛挫林，甾體類如氟替卡鬆）

21.喉用錠劑

【抗生素】

22.環丙沙星、盤尼西林、大環內酯抗菌劑、頭孢

【外用藥】

23.抗菌和類固醇霜

24.驅蚊劑

25.摩擦膏

【鎮靜劑】

26.咪達銼倫片劑

【可選】

自動體外除顫器（AED）

▍設備

  • 眼部護理套裝（眼墊、眼藥水、洗眼液等）
  • 基礎醫療設備（溫度計、聽診器、耳鏡）
  • 手術縫合套裝，石膏
  • 通用夾板
  • 粘性氈 / 泡沫
  • 冰袋
  • 保險套
  • 靜脈注射套裝
  • 柺杖

業餘球隊的隊醫需要準備什麼？

除了第一時間撥打 120，草根隊醫還可以發揮更大的作用。

理論知識：

接受醫學 / 急救訓練，越專業越好。

由經過醫學訓練的人來擔任隊醫當然最好。由普通球隊成員兼任的隊醫，建議去考一個急救證，並經常複習急救知識（參考：如何拿到急救資格證？ ），有條件就考 BLS ，參加紅會組織的急救員培訓也是不錯的選擇。

業餘球場上，需要隊醫現場處理的大多正是急救。注意，隊醫接受急救培訓的意義既在於實施急救，更在於提高其判斷力。以紅會的初級急救員爲例，課程主要教授心肺復甦和包紮。第一時間 CPR 的重要性不必多說。科學的包紮、止血和骨折固定，也有利於受傷的隊員更快康復，更快回到心愛的球場。

此外，如果感興趣，運動生理學和營養學知識也可多多涉獵。再次強烈安利 FIFA 的 F-MAC 足球醫學指南。

簡易藥箱：

按使用頻率排序，有經濟能力的可以向專業隊醫的藥箱看齊。

  • 冷卻噴霧
  • 冰袋
  • 雙氧水
  • 酒精棉片
  • 快速止血棉 / 止血帶
  • 棉花 / 籤、紗布、繃帶
  • 布洛芬
  • 按摩工具

其實業餘足球現場最理想的情況莫過於，擁有球場 + 隊醫的雙保險。常用藥物、擔架、AED 這些在攜帶和保管上有較高要求的物品，由球場提供再合理不過。

最後祝大家武運昌隆，業餘足球生涯華麗、健康、長長久久。
