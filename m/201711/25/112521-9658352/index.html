推薦 10 部適合年輕人看的電影，希望你喜歡

有哪些電影一定要趁年輕看？

以下分享的電影是站在我的角度上安利的覺得值得一看的電影，但也不是僅適合年輕人看，畢竟年不年輕，有時候僅僅只是一個心態問題~

ps：本回答已經儘量避開此回答下高贊答案裏所提到過的高頻率電影，力爭介紹一些其他被忽略的電影，豐富大家的閱片量~請大家笑納

1【心動】導演：張艾嘉 / 主演：金城武、梁詠琪、莫文蔚 / 豆瓣評分：8.2

由張艾嘉導演及編劇的一個有關於初戀遺憾的故事。香港的 1977 年裏，某天，17 歲的小柔在樂器店旁邂逅了浩君，她不小心冒冒失失的撞到樂器店玻璃門，看到在裏面專注試吉他的浩君，他倆四目相對，愛情開始的美好模樣，年少的悸動總是夾帶着驚喜。他們愛上彼此，在最少年青春的時候。他們一起淋雨，一起手牽手吃着路邊攤，在無人的巷口接吻，即使他們相約去外面小旅館過夜，卻也只是在旅館的房間裏相擁望天亮。在港口碼頭邊，浩君把小柔擁進自己的厚實外套裏，兩個年輕的靈魂，既熱烈又純粹。然而愛情裏也不全都是美好，雙方父母的插足讓這段感情舉步維艱，加上日漸增長的猜忌，誤解，以及小柔好友陳莉的插入，於是在一次大吵後，他們分手了。

再見面已經是多年以後，在東京偶遇的他們這次毫不猶豫的在酒店裏纏綿了一晚，把心裏多年來對彼此的情愛袒露相對。只是後面當離婚後的浩君向小柔求婚時，她拒絕了。遺憾一久也就失去期待的驚喜，而你缺席的那段時光已經由別人代替你幫我填補上了。電影裏最動情的地方就是多年以後的浩君在機場交給小柔一個木盒子，飛機上的小柔打開看，裏面是分開多年裏浩君拍的一些關於天空的照片，背面寫着：這就是我想你的日子，現在我把它們全部送給你。照片裏形狀各異的雲朵形狀，照片背面手寫了思念，日期從 1978 年到 1992 年。原來愛一直都常存心裏，只是時光已經回不去。

2 【阿飛正傳】導演：王家衛 / 主演：張國榮，張曼玉，劉德華，劉嘉玲，梁朝偉 / 豆瓣評分：8.5

英俊風流的旭仔阿飛（張國榮飾）是上海移民，他從未見過生母，自小由養母撫養大，因此長大後他對生命中遇到的每一個女人都表現得放蕩不羈，他先後與體育會售票員蘇麗珍（張曼玉飾）和舞女咪咪（劉嘉玲飾）同居，然後又相繼冷酷無情的和她們分手，她們給過他溫暖，但是那不足以讓他的心安定下來。阿飛從養母處得知，自己竟是混着西班牙和菲律賓貴族血統的私生子，所以他決定要找到生母，看看當初拋棄自己的生母究竟長得什麼樣子。爲此他隻身前往菲律賓，去到後生母毫無留戀的拒絕見他，最終他只能落寞離開，帶着遺憾和怨恨。

他把自己比喻成一隻“無根鳥”，只能一直不停歇的飛，停下來的那天就是死亡的時候。他從生母的居住地出來後在菲律賓生活了下來，後面與一直暗戀蘇麗珍的警察超仔（劉德華飾）在這異國他鄉相遇。不久，旭仔因爲買賣假護照而在一場毆鬥中身負重傷，最後時刻是超仔守着他，最終阿飛死在一列異國火車上，無根鳥終於落地

墨鏡王的經典作品之一，很值得細細回味，張國榮也憑藉這部電影攬獲了第 10 屆香港金像獎最佳男主角獎。

3【藍色大門】導演：易智言 / 主演：桂綸鎂，陳柏霖，梁又琳 / 豆瓣評分：8.3

17 歲高中生孟克柔（桂綸鎂飾）與林月珍（梁又琳飾）是無話不談的好友，月珍告訴克柔，說自己喜歡上了一個叫張士豪（陳柏霖飾）的男生，月珍給孟克柔看自己偷偷蒐集的有關張士豪的一切，叫她戴上自己偷拍張士豪的照片扮演他，叫她跟她一塊去偷看張士豪游泳，叫孟克柔去幫她表白，叫孟克柔去幫她送信，卻在她給張士豪的信裏留了孟克柔的名字。導致張士豪以爲喜歡自己的是孟克柔，所以也常常藉機會去接近克柔，製造跟她在一起的機會

張士豪在學校是個很受歡迎的男生，帥氣陽光，善良單純，最大的願望就是拿到游泳比賽冠軍和成爲孟克柔的男朋友。隨着他和孟克柔的相處，他們成爲了朋友，在一次交談中，克柔跟他袒露了自己的心事，袒露了自己對月珍那懵懵懂懂的同性愛戀。而後張士豪跟孟克柔說，希望多年以後如果她喜歡男生了的話，一定要第一個跟他說。結尾孟克柔望着張士豪騎着單車騎前面飛馳，風吹起他的花襯衫，她心裏獨白說：“雖然我閉上眼睛仍看不見自己，但卻能看見你”。算是預示着他們之間的另一種可能性，因爲他們的青春還在繼續。

4【新不了情】導演：爾冬升 / 主演：劉青雲，袁詠儀，劉嘉玲 / 豆瓣評分：8.

一個堅強樂觀的女孩鼓勵一個失意的男人的故事。

阿杰（劉青雲 飾）是一個抑鬱不得志的音樂人，他爲了堅守原則放棄了優越的生活和女朋友，生活潦倒不得不獨自住進了一個破舊的單位內，終日都沉默寡言。此時他也遇到了在廟街街頭藝人的女兒阿敏（袁詠儀 飾），阿敏雖然自小便患有骨癌，但她是個非常開朗樂觀的女孩，會自找樂趣，穿裙子時會害羞，塗完口紅會不敢張嘴說話，時常呵呵大笑，開導起阿杰的時候大道理一堆堆，她對生命充滿希望的態度感染了阿杰，令他找回了失去的自信，慢慢的，在事業上阿杰也看到了曙光。兩人相處中漸漸對彼此產生情愫，就當一切看來會朝美好方向發展的時候，阿敏的病情卻日漸惡化，無數次化療的她望着一撮撮掉落的頭髮，她終於失去了往日的勇敢及堅強，變得暴躁和敏感。而阿敏的母親爲了阿敏的治療也打破自己之前的諾言，再次登上劇院唱戲。最終，阿敏沒有撐住，她再沒來得及吃上阿杰買回來的鉢仔糕。

5【菊次郎的夏天】導演：北野武 / 主演：北野武，光口雄介，岸本加世子 / 豆瓣評分 8.7

菊次郎（北野武 / 飾）是一個遊手好閒的自由職業者，靠妻子維持着家庭生活。鄰居三年級小學生正男（光口雄介 / 飾）和奶奶生活在一起，恰逢暑假來臨，正男拿着母親從遙遠遠方寄過來的信，心裏突發奇想要來一場冒險旅行。菊次郎的妻子（岸本加世子 / 飾）由於不放心，就讓菊次郎一起同行，並給了他一些旅費。結果第二天，菊次郎就把旅費都輸光了，最後只得和正男一個步行上路。菊次郎看似吊兒郎當的外表下其實也有一顆溫柔的心，在來到正男母親的住址後，當正男看到他的母親正好穿着圍裙從屋裏走出但是還看到母親身邊還有另外一個男生和小女孩，正男意識到自己是被母親拋棄了，菊次郎不忍看正男傷心，於是編造了一個美好的謊言。同時，菊次郎也做出了一個決定，他打算回去看看自己那在養老院的母親，等到那裏的時候，他沒有驚擾母親媽媽，而是遠遠觀望然後平靜的離開。一路上菊次郎和正男遇到形色各異的陌生人，也碰撞出很多歡快的火花，一路上善良又可愛的陌生人們都在盡力在給予正男一個最歡快質樸的童年回憶。

這是一部北野武自編自導自演的電影，一部關於童年和成長的溫情公路電影。畫面清新再配上久石讓的音樂，還有歡快的劇情，滑稽感動又滿滿的溫馨，很治癒。

6【步履不停】導演：是枝裕和 / 主演：阿部寬，下川結衣，江原由希子，高橋和也 / 豆瓣評分：8.8

位於偏遠小鎮的橫山一家，生活平靜祥和。父親恭平（原田芳雄 飾）是業已退休的醫生，卻時時牽掛小鎮診所內的事務。長子純平繼承了父親的事業，卻在 15 年前爲救落水兒童而溺水身亡。次子良多（阿部寬 飾）與父親意見相左，執意前往異地當起了繪畫修復師，然而多年的打拼換來的卻是失業，生活困頓的良多與帶着孩子的由香裏（夏川結衣 飾）結婚，相互扶持，繼續生活。又逢純平的忌日，良多和姐姐千奈美（YOU 飾）帶着家人分別趕回家中參加此次聚會，平淡的家裏因爲此次相聚變得再次熱鬧起來，但是熱鬧中又伴隨着生疏，自從長子溺水身亡後，老父親忽略對次子的關愛，以至於良多心裏生出積怨已久的情感隔閡。最後隔閡慢慢消除後我們能看到的是一個其實也無非就是一個溫情的老父親，一個慈愛的老母親和一個不善於表達的兒子的故事。

這是一部探討死亡和救贖的電影，一部關於死亡意外是如何改變一個家庭的電影。一如既往帶着是枝裕和導演的平和，電影劇情雖然沒有什麼起伏，不溫不火，但隱隱能讓人感覺到平底下的絲絲暗涌，但是導演沒有讓它爆發，而是讓它在這慢慢幻化進父母深沉而又隱忍的愛裏。這樣的故事，既平淡但又最動人。

7【生命如此美好】導演：馬切伊佩普 / 主演：大衛·奧格尼克，多蘿塔·科拉克 / 豆瓣評分：8.4

一部根據真實事蹟改編的電影：Mateusz 是一個先天殘障男孩，生下來殘疾並不能說話，從出生的那一刻起，他就註定了這一生的路要比別的孩子走得更加辛苦艱難。痙攣似的四肢、扭曲變形的手指、僵硬的脖子、只能發出沉重的呼吸卻無法與人溝通交流，無法表達自己的思想……每次看到他焦躁不安地扭動身軀，那雙掙扎着想要說話的眼睛都讓人感到揪心的痛。其實他什麼都知道，對外界的事物他也有自己的想法，但是大家都以爲他是像植物一樣什麼都不懂的，後來他被送到精神病院，被醫生髮現他聽得懂並教他怎麼表達，在最先發現 Mateusz 有強烈的表達意願的語言康復老師日復一日的幫助下，Mateusz 終於藉助語言圖冊鏗鏘有力地說出了這句話，“我不是一顆蔬菜！（No I Vegetable）”，這是 Mateusz 26 年以來表達出的第一句話。

電影描述了一個不幸但卻溫暖的故事，讓我們看到生命的堅韌，健全的我們又有什麼理由去抱怨生活呢。

8【Begin Again 再次出發之紐遇見你】導演：約翰卡尼 / 主演：凱拉·奈特莉，馬克·魯法洛，亞當·萊文 / 豆瓣評分：8.5

格雷塔（凱拉·奈特莉 Keira Knightley 飾）拋棄了一切跟隨着男友戴夫（亞當·李維 Adam Levine 飾）來到美國紐約實現他們的音樂和愛情夢想，沒想到曾經幻想過的美好一切化作泡影，在獲得了更好的工作機會後，戴夫果斷拋棄了格雷塔，留下她一人苦苦在底層掙扎。 丹（馬克·魯法洛 Mark Ruffalo 飾）最近的生活也不好過，他發現自己無法從思念前妻的感情中走出來，與此同時，他的女兒維奧萊特（ Hailee Steinfeld 飾）正處於一生中最叛逆的階段，父女關係差到極點。憤悶的他有一天在酒吧裏遇到了偶然登臺演唱的格蕾塔，他堅信自己找到了一顆冉冉上升的新星，於是開始召集人馬組建樂團做唱片，在城市街頭演唱錄音，採集城市的聲音，最終如願將唱片做了出來，獲得成功，雙方都如願實現自己的夢想。

約翰卡尼的又一作品，雖然又是老套的永不放棄，追尋夢想的故事，卻還是滿載着正能量。特別推薦電影裏的歌曲，真的很值得一聽。

9【辯護人】導演：楊宇碩 / 主演：宋康昊，吳達洙，今英愛 / 豆瓣評分：9.1

1978 年，只有高中學歷的宋佑碩（宋康昊 飾）通過多年的艱苦努力，終於通過司法考試，並在成爲法官後很快轉行成爲一名律師。他敏銳地從最新政策中嗅到商機，以不動產代書業務起家。雖然被同行譏諷爲隨時隨地派發名片的夜店小弟，但佑碩不以爲忤，一步一步朝着心中的目標邁進。他依靠賺來的錢讓妻兒過上富足的生活，也還了七年前在飯店大嬸那裏欠下的良心債。進入 20 世紀 80 年代，韓國民主化鬥爭愈演愈烈。佑碩全然不管窗外事，關起門來繼續賺錢，然而社會的鉅變已經不容他置身事外，飯店大嬸的兒子樸鎮宇因從屬的釜山讀書聯合會被控爲左翼社團而遭到逮捕，更受到殘酷的虐待和不公的指控。大嬸求宋佑碩幫自己的兒子辯解，雖然愛錢卻更重感情的佑碩，就這樣走上了爲民主辯護的道路。面對着勝算不大的官司和官官相護的嘴臉，有勇有謀的佑碩與對方鬥智鬥勇，他逐漸在這個案件中領悟了憲法存在的真諦，他找到了值得自己爲之堅持奮鬥的正義。

10【人狗奇緣】導演：樸銀炯，吳達均 / 主演：俞承豪，金香起，安吉江 / 豆瓣評分：8.3

11 歲的哥哥燦伊（俞承浩 飾）一直都像個大人一樣，無微不至的照顧着妹妹小伊（金香奇 飾）。兄妹倆人一直都相信，只要他們乖乖聽話，總有一天，離家出走的媽媽會回到他們身邊。 這一天，小伊生日到了，爲了滿足她的心願，哥哥送了一隻剛剛出生小狗送給她，小伊認爲這是媽媽送給她的禮物，所以取名爲“心意”。有了“心意”的陪伴，燦伊不在家的時候，小伊有了玩伴。三個小傢伙像一家人一樣，快樂的生活着。轉眼“心意”長大了，在一個陽光燦爛的冬日，“心意”拉着冰車，帶着小伊在結冰的江上快樂的玩耍，不料發生意外，小伊掉進冰窟窿裏面，燦伊把小伊意外的過錯歸咎在“心意”的身上，於是燦伊拋棄了“心意”離家出走，準備踏上火車去尋找媽媽，但“心意”一直跟隨在後面，任憑燦伊的打罵，依然不離棄，最後火車開走了，“心意”也沿着軌道追跑着，最終當它在軌道旁看主人燦伊被壞人欺負時，它義無反顧的衝上去保護了主人，最後它被打得奄奄一息，再加上它本來就患有眼病，醫生告訴燦伊，心意已經時日無多。此時燦伊終於明白了“心意”對自己的忠誠和愛，最後當狗狗永遠的在長椅上永久的睡着後，燦伊很痛苦很悔恨，失去妹妹和“心意”的他現在真的只剩自己了。

特別催淚腺的一部電影，狗狗的忠誠讓人動容，但願大家都能做個有情有義有愛心的人。
