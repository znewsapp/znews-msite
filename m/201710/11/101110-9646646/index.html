做腦外科手術時，病人的大腦會像豆腐似的被切來切去嗎？

專業對口的問題啊～怒答一發

如滷煮所說，腦子的質地確實像豆腐哦，形容一下大概是如內酯豆腐般細膩的北方豆腐。

神外手術的器械不同於其他外科，主要用的是雙極電刀（像一把槍狀鑷子）和吸引起（管壁帶一個側孔可以通過手指按壓調節吸力）。上圖！

其他的還有顯微剪刀、持瘤鑷子、顯微鑷子等等。因爲現代的神外手術提倡顯微操作（上世紀 90 年代前都是肉眼直視下操作，損傷很大），所以器械大多都是槍狀的以避免阻擋顯微鏡下視野。特殊的手術還會有特殊器械，比如我們常做的腦血管病中，動脈瘤和腦血管畸形還要用到動脈瘤夾（幾乎純鈦質地的小夾子）

神經外科手術主要通過雙極燒灼 - 配合吸引器鈍性分離 - 顯微剪刀剪斷結締組織（燒灼後的血管和腦膜）這一系列動作完成。

那麼問題來了，腦子真的是像切豆腐那麼切嗎?

當！然！不！是！

豆腐可以切成菊花，但腦子要被切成那樣就只能下鍋了（誤）！

手術原則是儘量保護正常腦組織（外傷的腦挫裂傷除外）。所以淺表病變一般我們都直接幹掉，少碰腦子。

那麼深部怎麼辦呢?

通常我們會利用大腦的天然溝裂，腦溝就是腦子一條條凸起之間的縫隙，但下面其實還是皮層，所以即使從腦溝進入也會有損傷。所以我們常用的包括側裂、縱列、小腦幕、顱底（腦子與顱骨間的空隙）。這些腦裂是腦葉之間自然存在的，只有些血管和顱神經走行，這些都是有彈性的，禁得住輕微牽拉，暴露的空間足以手術。

藍箭頭爲縱裂，紅色爲側裂，可見其實可以深入挺深的。

那麼總有這種腦裂夠不到的，或者不合適的。怎麼辦呢?那就選腦溝造瘻進去（就是樓主說的“切開”，其實這個也是通過我上面說的雙極完成的）。那麼如果要造瘻的話我們一定會選最短路徑，以最大程度保護腦組織。以前的做法是通過 ct 或者 mri 來手工定位，通過一些標誌線，在體表上畫出最短路徑下的病竈投影，手術開進去。這個是個技術活，也是每個神外老大夫的看家本領。我們年輕小大夫大多需要藉助捲尺完成，老大們直接目測就八九不離十。

補充一些用於定位病竈和功能區的高（黑）科技：

1.血管病定位的小天使——術中 B 超

B 超大家都做過吧，是利用機械波在不同質地的物質間存在反射差異，從而在無創的情況下確定深層組織信息。B 超機的樣子基本都一樣，大家有興趣可以花 42 塊錢掛號（北京醫保報銷 40 塊）約個腹部 B 超看看，順便也體個檢嘛。

最開始的 B 超很苦 B，是一維線性的，回來的結果是探頭下一條線上的回波強弱，只能根據強弱來猜，具體組織結構看不到。我目前還見過的唯二兩個一維線性超聲波儀器，一個是病房裏一臺老掉渣的經顱 B 超機和我爹用來機械探傷的玩意兒。（實在找不到圖，可以參考示波器）

後來 B 超有了二維圖像，世界一下子就光明瞭

網絡圖片，侵刪。

可以看到肚子裏小 BB 的形態。這個時候基本上已經具有神經外科應用的價值了，比如探測顱內血腫、腫瘤（有一定侷限性），只要是組織構成異常的都可以。然而對於腦血管病中的腦動靜脈畸形，這還是不夠的，因爲這種病是短路的動靜脈摻雜在正常的腦子裏，從組織構成上看就是腦組織 + 血管，較小（難以定位和發現）的病竈用這個二維 B 超可能看不出來。這時候，多普勒（Doppler）影像解決了這個問題，這個技術可以測量探頭扇面下的血流速。如同路邊的交警叔叔測你超速一樣，用探頭原地掃描一個心動週期，下面血管內血流速度一目瞭然，病竈因爲具有特徵性的血流狀況，所以無處遁形。於是就可以根據 B 超探頭的位置和角度造瘻進去，直搗病竈。由於 B 超太常用了，所以沒想過照相。（捂臉）

另外，去年我們辦過一個術中 B 超導航的學習班，B*K（一個 B 超牌子）與 Bra***ab（上面提到的導航廠家）合作，把 B 超和神經導航整合在一起了。就是說可以把 B 超掃描到的無數二維平面影像整合到 MRI 或 CT 構成的三維影像中。

那麼這個技術有什麼意義呢？畢竟 B 超那麼難看。意義在於，MRI 或 CT 圖像再好看，也是術前的。術中隨着顱骨和硬膜的開放，腦組織本身就會有位移，造成導航不準（漂移），這時候利用 B 超可以進行修正。此外，切除腫瘤或者腦血管畸形後的腦組織也是會位移的，B 超無疑是修正這種漂移的最便捷手段。

2.神奇的電生理——術中電刺激與電生理監測

首先說基礎——電生理監測

這個是在病人全麻後，在頭皮和四肢插入針式電極（讀起來很疼對不對？我們看着也很疼啊，但病人全麻啊，扎的時候已經睡着了，清醒之前已經拔掉了，大概只會莫名其妙地覺得手疼腳疼（捂臉））。在中央前回附近的電極發放電刺激，監測四肢肌肉電活動叫做運動誘發電位（motion evoked potential）；四肢電極發放電刺激監測中央後回腦電活動稱爲體感誘發電位（sense evoked potential）。這個是監測腦正常功能的最基本的兩個方式，簡單理解就是兩條路一個輸入、一個輸出，兩條路都通的話，神經的高速公路就是完好的，細菌就能以每公里三百公里的時速。。。。啊呸，不賣柺了~術後肢體活動就不會受影響。

那麼術中電刺激是什麼呢？就是再弄一個無菌的電極在術區，在開顱後分辨一下眼下暴露的是哪個腦回（可以通過肉眼，也可以通過導航），然後把電極放在上面予以微弱的電流刺激（mV 級），通過觀察 MEP 和 SEP 變化以確定是否抑制局部大腦皮層功能，確定這部分皮層的功能，一般用於功能區的精確定位和驗證。

這是去年病區做的一例腦動靜脈畸形（AVM），病竈和中央前回（運動區）距離很近，通過術中電刺激找了一塊不影響運動和感覺的皮層造瘻切進去。術後病人走着出院噠~ (•̀ᴗ•́) ̑̑

作爲術中電刺激的延伸，還有一類鬼畜操作叫做術中喚醒。。。當然這個操作存在一定的倫理學爭議，畢竟你一覺醒來發現別人在玩你的腦子，這個感受不會太好（儘管你知道自己被送去手術了）∑(っ °Д °;)っ

這個操作中，會讓病人在術中醒來，予以相應的運動、語言或視覺皮層電刺激，看病人時候還能進行肢體活動、聽說讀寫和看的活動。一般是用在膠質瘤一類病人中，因爲這類病變邊界不清晰，主刀即使經驗豐富也經常不能很確定是否切除乾淨，通常會多切一些。那麼多切的這部分是不是功能區呢？不要緊，叫醒驗證一下咯~ 不過說來簡單，術中喚醒實際操作起來對手術團隊要求很高的，切莫輕易嘗試哦。

3.電磁不分家——經顱磁刺激（Transcranial Magnetic Stimulation，TMS）

電生磁，磁生電。既然皮層是電活動，那麼我不開顱，直接造個局部磁場也可以影響皮層電活動咯~邏輯就是這麼簡單，也確實就是這樣。

TMS 是近些年發展蓬勃的一種無創皮層刺激方式，用線圈產生三維定向的磁場（可以刺激三維空間內的一個很小的區域，可以理解爲“點”），刺激局部皮層，產生和術中電刺激一樣的生理效果。一般 TMS 都是在神經導航提供的位置信息的輔助下進行的。
