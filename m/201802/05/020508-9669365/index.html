Facebook 用戶時長減少了 5000 萬小時，扎克伯格對此還挺滿意

Facebook 用戶每日使用時長少了 5000 萬小時，意味着什麼？

Facebook 的用戶時長減少了，扎克伯格爲什麼還挺驕傲？

重視用戶粘性，是 Facebook 和微信在增長近乎觸頂後作出的決定，也是在普通用戶開始反思科技和互聯網的弊端之後，科技公司們必須作出的迴應。

過去一年對 Facebook 而言非常特殊。通俄門、直播自殺和假新聞等等事件讓年初就準備好遊歷全美 50 個州的公司 CEO 馬克·扎克伯格忙到了焦頭爛額。他也許從沒想過，Facebook 會惹上這麼多的麻煩，更不要提這些麻煩事恰恰是與他對 Facebook 平臺的美好設想完全相悖。

正因如此，2017 年 Facebook 的財務表現才更加讓人關心。在財報發佈之前，就有不少媒體和分析師唱衰的聲音，在財報發佈之後沒多久，Facebook 的股價就因爲一些不太好看的數據而應聲下跌 4.5%。

是哪裏出現了問題呢？

從財報數據來看，Facebook 2017 年第四季度和全年的總營收同比漲幅均達 47%，超過華爾街分析師的預期，即使在特朗普去年年底的稅改之後上交了 22.7 億美元稅款，Facebook 也仍然保持着利潤的大幅增長。

但如我們過去多次提起過的，Facebook 在 2017 年裏經歷了不少的波折，這也是財報發佈之前許多人唱衰 Facebook 的原因。在財報中，扎克伯格自己也說道，「2017 年對 Facebook 而言是強勁的一年，但也是艱難的一年。」雖然股東和分析師們仍然盯着營收和利潤的數據，但 Facebook 眼下卻有着更加複雜而棘手的問題需要解決。

我們首先來看一下財報裏比較重要的一些數據：

  • 總營收：第四季度同比增長 47%，年度同比增長 47%；
  • 淨收入：稅改原因導致第四季度同比增長 20%，全年同比增長 56%；
  • 稀釋每股收益：第四季度同比增長 19%，年度同比增長 54%；
  • 日活：日活躍用戶 14 億，同比增長 14%；
  • 月活：月活躍用戶 21.3 億，比去年增長 14%；
  • 移動廣告營收佔比：移動廣告營收在廣告總營收中佔比 89%，高於去年同期的 84%。

單從這些數據來看，Facebook 的總體情況似乎一片大好，排除稅改增加的成本，公司在營收和用戶增長方面仍然保持着傲人的成績。但如果要與去年的增長情況相比，似乎就沒有那麼樂觀了。

2016 年，Facebook 日活躍用戶和月活躍用戶相比 2015 年分別增長了 18% 和 17%，遠高於今年的 14%。在那一年的年終報告上，扎克伯格曾表示：「我們連接世界的使命變得前所未有的重要。」可在 2017 年，扎克伯格似乎並沒有集中所有的精力去吸引新的用戶。

大好形勢下的兩個小「隱患」

不僅僅是活躍用戶的增長速度減緩，在 Facebook 亮眼的財報數據中，還有兩個更讓股東和分析師們擔心的「小數據」：用戶平均每日使用時長減少了 2.14 分鐘，美國和加拿大的月活躍用戶從 1.85 億下降到 1.84 億。

「在過去的一個季度裏，我們對病毒傳播的視頻做出了一些調整，以便於人們更好地使用他們的時間。總的來說，我們削減了用戶在 Facebook 上消耗的時間，大約每天 5000 萬個小時。」扎克伯格在財報中說道，他不僅沒有迴避這一數據，反而表現出非常滿意的態度。

5000 萬個小時平均到每個月活躍用戶身上，也僅僅只有 2.14 分鐘，大約佔據每個用戶日平均使用時長的 5%。在許多分析師眼中，這 2.14 分鐘代表着一個非常危險的趨勢。Facebook 最主要的營收來源於廣告收入，而廣告收入恰恰取決於平臺上用戶的活躍度。因此，用戶使用時長的減少，通常意義而言，會對 Facebook 的廣告收入造成直接的打擊。

另一個不大好看的數字——美國和加拿大日活躍用戶的減少，同樣加深了分析師們的焦慮。長期以來，Facebook 在主戰場北美一直保持着穩定的增長趨勢，雖然近幾年的增幅不比亞洲等尚未飽和的市場，卻也從未出現過下降的趨勢。

Facebook 財報中日活躍用戶的部分數據；圖源：TechCrunch

在 2017 年，Facebook 因爲平臺上假新聞氾濫和政治廣告等問題遭受了社會各界的諸多批評，同樣遭遇的還有 Twitter 等社交媒體。這一年裏社交媒體的公信度大大降低，很多人不僅開始質疑這些平臺上信息的真實度，更對社交媒體本身的弊端有了許多反思。

Facebook 和扎克伯格本人當然也意識到了這點。「在 2018 年，我們將重點確保 Facebook 不僅有趣，還要對人們的生活和社會有益。」在個人的 Facebook 主頁上，扎克伯格如此寫道，「我們正鼓勵人們彼此之間更有意義地連接，而不是被動地消費內容。」

扎克伯格在個人 Facebook 主頁上對財報的評價

Facebook 的「自救」

不同於去年的「連接更多人」，Facebook 今年的想法發生了改變，用很多外媒的話說，Facebook 正在「自救」。

從去年年底算起，Facebook 首先調整了信息流的算法，將信息流中媒體內容的佔比從 5% 削減到 4%，增加好友動態的比重，讓平臺屬性迴歸社交。

隨後 Facebook 推出平臺上的媒體評級系統，將綜合主觀評分和用戶評價來爲媒體劃分等級，以便於爲優質的媒體和內容提供更多的展示機會。

前兩天，Facebook 又宣佈封禁所有虛擬貨幣和 ICO 相關的廣告，以防止用戶在平臺上受到欺騙，造成經濟損失。

總體上看，雖然仍有不少的爭議聲，但我們能從這些大刀闊斧的改變中看到 Facebook 和扎克伯格的思考與決心。「短期內，我們想做一個重新的平衡，但是長期來看，用戶會覺得在這個平臺上找到了很多有意義的東西，而不是完全被動地去消耗一些內容。當你找到有意義的東西時，你會願意回來。」這是 Facebook 工程總監範鵬在接受極客公園採訪時表達的觀點。

Facebook 工程總監範鵬接受極客公園採訪

表面上看，海外地區越來越成爲 Facebook 內部的「增長黑客」們重點發力的市場，他們仍有能力讓產品的用戶數量保持穩定增長。在這樣的條件下，Facebook 也開始有了餘力，來思考如何彌補「野蠻增長」可能會帶來的一些問題。

第三個指標

前面談到用戶 Facebook 的營收主要來源於廣告，而廣告又依託於用戶在平臺上的表現，用戶越活躍，廣告展示的機會和吸引到的注意力越多，Facebook 的營收增長也會越明顯。

在這裏，用戶的活躍並不僅僅體現在人數和使用時間上，Facebook 已經成爲世界範圍內一家獨大的社交平臺，用戶數量的增長遲早會見頂。而如上面所說，惡性追求用戶使用時長會引發各種各樣的社會問題，這時第三個因素就顯得格外重要：用戶粘性。

如產品經理張小龍所說，「用完即走，走了還會回來。」扎克伯格的想法其實也是一樣，留住用戶不一定意味着要讓用戶無休止地使用自己的產品，只要用戶喜歡上產品，即使離開，也會回來再次使用。這種對用戶粘性的重視，是微信和 Facebook 兩者在增長近乎觸頂後產生的思考結果，也是過去這一年中，普通用戶開始反思科技和互聯網的弊端之後，科技公司們必須要作出的迴應。

「如果你把核心的事情做好，用戶的增加、銷售收入的增長應該會隨之而來。」範鵬如此解釋道。

反觀 Facebook 的財報數據，即使用戶增長速度有所放緩，使用時長略微減少，但 Facebook 日活躍用戶佔據月活躍用戶的比重卻沒有發生任何改變，與前幾年一樣，同樣保持在 66% 左右的高位。

也就是說，即使用戶花的時間沒那麼多了，也沒有任何趨勢顯示他們會離開 Facebook，從這一數據來看，它將讓 Facebook 在文化和政治問題上變得更輕，更多去承擔客觀的工具身份，但對廣告營收卻沒有明顯的影響。

當然，去年的財報數據只能看出 Facebook 在遭受一系列打擊之後緊急調整的結果，在今年扎克伯格卯足了勁「改革」之後，更多重視用戶粘性的產品思維最終會收穫怎樣的結果，還要看未來 Facebook 的表現。

但至少，在短暫的下跌之後，Facebook 的股票開始反彈，目前，盤後已高於財報發佈前股價。
