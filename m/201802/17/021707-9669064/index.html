年度熱門 · 已婚男女可以和伴侶之外的異性單獨吃飯嗎？

美國 53 % 的女性和 45 % 的男性都認爲這麼做不合適。

《紐約時報》請調查公司 Morning Consult 問 5,000 多個美國人下面這個問題：

  • （如果調查對象是女性）單獨和不是你丈夫的男性開會 / 坐車 / 吃午飯 / 吃晚飯 / 喝酒合適嗎？
  • （如果調查對象是男性）單獨和不是你妻子的女性開會 / 坐車 / 吃午飯 / 吃晚飯 / 喝酒合適嗎？

如上圖所示，多數人認爲和妻子以外的女性吃晚飯、喝酒都不合適，且女性中認爲“不合適”的比例比男性高。

38 % 的女性和 29 % 的男性認爲“坐一輛車”不合適，25 % 的女性和 22 % 的男性認爲“工作開會”也不合適。

由於調查是商業公司開展的，所以抽樣方法、原始數據都沒有公開，我們也不知道這個問題是否只問了已婚男女。不過 Morning Consult 在業內很有名，信譽是有保障的。更細節的數據，他們提供了下面這兩張圖：

共和黨、福音教、沒有大學文憑的調查對象，說“工作開會不合適”的比例更高。

在女性受訪者裏，年紀越大（橫軸右側）、宗教信仰越強烈（實線），更有可能認爲“和丈夫以外的男性單獨吃晚飯 / 開工作會議”不合適。

--

今年三月，《華盛頓郵報》寫了一篇關於副總統彭斯妻子的長文，提到 2002 年彭斯曾對記者說，他從不和妻子之外的女性單獨吃飯，從不一個人參加提供酒精的活動。

文章發表後引起美國社會的激烈爭論，也纔有了本文開頭的那個調查。

《華盛頓郵報》和上述調查的評論區都非常熱鬧。幾個高讚的評論大意如下：

  • 這是美國還是沙特？
  • 人類是隻有原始性慾的動物嗎？一男一女談工作，大家就認爲他們一定會發生什麼，真悲哀，簡直是對人類的侮辱。
  • 作爲一個男老師，我也很無奈地選了“不合適”。以前我還經常請學生來家裏做客，但現在一律只在課上交流，不論男女。因爲我已經被很多人提醒過，和學生接觸是冒不必要的風險。萬一被告性騷擾，對我名譽的打擊是毀滅性的。
  • 我是個女的，我也不和男同事單獨見面，因爲之前被性騷擾過。我也不會表現得太友好，免得引起男同事誤會。
  • 我是個女醫生 / 女記者 / 女博士 /... 做我們這行的不和男性單獨見面，那工作都開展不了，也別想晉升了！
  • 如果副總統單獨和女性待在一起都怕“忍不住誘惑”，他憑什麼領導我們國家？

--

這份調查是過去所有民調裏對我衝擊最大的，完全顛覆我對美國社會的認知。從媒體和文藝作品裏看到的美國，是在兩性問題上非常“開放”的。而我過去和現在老師、老闆，90% 都是男性，如果不和他們單獨接觸，根本無法完成最基本的任務，更別說很多時候還得加班、公司聚會。

調查結果出乎你的意料嗎？如果讓你猜，美國人有多少覺得吃晚飯不合適，你會猜到“多數人”嗎？如果讓你猜，美國人有多少覺得坐一輛車、一起開會不合適，你會猜到 “三成、四成” 嗎？

女性比男性選“不合適”的比例更高，這我也沒想到。問題問的是“你和不是你丈夫的男性吃飯合不合適”，而沒有問“你的丈夫和不是你的女性吃飯合不合適”，所以女性在回答這個問題的時候，代入的應該是自己的情況。我覺得唯一能說得通的解釋，就是美國女性信仰宗教的比例更高，所以選“不合適”的比例比男性高。從美國網站的評論區來看，可能還有女性選擇“不合適”是出於安全考慮，怕和男性獨處會有危險。不過我們無從知曉因爲這個原因而選擇“不合適”的比例有多高。

--

社會科學研究很多時候“發現”的東西都是“顯而易見”的，比如女性生育會影響長期收入、社會存在階級固化。

但是“顯而易見”只是回過頭來看才“顯而易見”。這份調查充分說明了實證研究的重要性。如果我以爲“美國人都特開放”，就不做這個調查，那根本不會知道我以爲“顯而易見”的結論其實是錯的。
