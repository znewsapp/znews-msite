關於汽車，普通人有哪些常見錯誤認知？

有很多人認爲高端品牌等於質量好，甚至等於這臺車一切都好。這就像認爲阿瑪尼比優衣庫的 T 恤不容易起球，或者 LV 的包比宜家的購物袋更結實。

有很多人認爲 Mini 和 Smart 是高端品牌。因爲他們的母公司分別是寶馬和奔馳。但他們不知道菲亞特和法拉利也是一個公司的啊？

很多人認爲中國特供就一定是偷工減料或者降低標準或者瞎搞。

很多人認爲原汁原味的全球車纔可能是好車。

假設 ABCDE 是同價位的類似車型，有很多人覺得 A 汽車秒殺 BCDE，有很多人覺得 B 汽車秒殺 ACDE。這種爭論的個人意義遠大於公共意義。這就像五個高考總分都在 500 分左右的學生，有人數學比較強，有人英文比較突出。很多人都只是站在個人的角度，把汽車強行分了三六九等。其實汽車是一個高度競爭的行業，同價位同類型的產品，互相總體水平差異不大。真正差異大的是個體消費者的特定需求。比如你有二胎你需要空間大，他單身愛 YP 所以喜歡拉風外形。

接上面一段：真正有操守的車評人，無論國內國外，一般都比較平和。誇不會誇上天，貶不會踩入地。很少說一個車就比同級別的車好到哪裏去。逛論壇的車迷更有可能爆出非黑即白的鮮明觀點。

專業人士中，只有戲劇娛樂節目工作者（如 Top Gear）或者需要吸流量造個標題黨的自媒體纔會觀點走極端。

其實保修期內不在 4S 店內保養，而是自行選擇合規的保養方式，並不影響質保——但大家是萬一出了問題，免不了去跟 4S 店爭吵一番。所以這個招數適合對汽車車輛質量很有信心、或者對自己吵架能力很有信心的人。

怠速啓停裝置真的能省油，而且不傷害發動機。但很多中國人本能的認爲頻繁點火費油 & 傷發動機（因爲評論區很多人提到會傷電池，特地說明：電瓶也沒問題，因爲這種配置的車的電瓶和傳統電瓶不一樣，是特別設計的）——汽車工程師真的不是文盲或者智障。

有很多人認爲開會會加塞、會亂變道、會在高速上不斷繞樁超車，是一種值得驕傲的技能。

很多人不知道什麼時候該用遠光燈，什麼時候該用近光燈。

很多人知道如果高速限速 120km/h，你就不應該開的比 120 塊。但很多人不知道，在這個情況下，你也不應該開的比 120 慢多少。

很多人不知道在高速公路上，當你的速度不夠快的時候，就別 TMD 佔在左側快車道。

其實在十字路口，紅燈變綠的時候，最高效合理的操作是：汽車全速起步，迅速衝過路口。但可惜，很多司機不理解這對公共交通效率的價值。或者，知道了也不敢這麼做——因爲很多行人、自行車和電瓶車連什麼是紅綠燈都不知道。

副駕駛不用系安全帶 / 後排不用系安全帶 / 出租車不用系安全帶 / 親人開車不用系安全帶 /……

事實：你坐過山車怎麼知道繫上安全帶呢？

-

地板油 / 高轉速傷發動機 / 轉速上 3000 會報廢

事實：發動機可靠性測試時的工作狀況比你踩兩腳地板油嚴苛多了，只要你不是冷車啓動就地板油，開到報廢發動機都沒事。

-

彈射起步傷車 / 彈射一定次數車就報廢了

（本條僅限於兩驅車型，全時四驅車型因爲機械結構的特性，頻繁彈射確實會損傷傳動系統。多謝@Taylor 提醒。）
事實：手動擋車彈射起步除了對離合片磨損比較明顯外，其餘和正常駕駛差別不大（損傷肯定有，車輛一旦開始運轉就會產生機械損傷，暴力駕駛對車輛的損傷確實比溫柔駕駛更大，但不會達到“傷車”的程度），有經驗的車手在彈射起步時爲了防止輪胎突破抓地力極限，還會將發動機轉速控制在一個較低的範圍，這種狀況下車輛負擔甚至小於地板油狀況。

而自動擋車都帶有轉速保護，油門踩進發動機裏也只給你 2000 出頭的轉速，想傷車都不給你機會。

至於彈射到一定次數就報廢，多半是 GTR 梗的訛傳。事實上 GTR 也只是連續彈射多次後電腦會限制你使用彈射功能作爲變速箱自我保護，並不會報廢。

-

延伸出另一條錯誤認識：

彈射起步＝燒胎起步 / 地板油彈離合起步

（感謝@鼯鼠的提醒）

事實：彈射起步是一種比較進階的起步技巧，具體操作是在車輛起步前將發動機轉速提升到一個較高的水平，從而獲得更大的馬力與扭力輸出。

但並不是轉速越高越好，過高的轉速會使傳遞到車輪上的扭矩過大，造成輪胎突破自身抓地力極限，發生俗話說的“燒胎”現象。雖然看起來很酷炫，但在起步時發生燒胎只會讓車輛加速變慢，並且增大操控難度。

想看真正彈射起步效果，可以上汽車之家搜測試車輛加速性能的視頻，具體過程就是

——原地拉轉速到 2000～4000 轉左右——鬆離合 / 鬆剎車——車頭一擡 / 車尾一墩——伴隨着短暫的輪胎哧哧聲——慢悠悠地開出去了——

沒錯就是這麼無聊！電影和動畫裏嗷嗷叫着拖着滿屁股白煙衝出去的橋段都是假的！

（另外一點延伸：燒胎對於提升加速性能確實有意義，但不應在起步時，而應在起步之前。其目的是通過摩擦來提高胎溫並使輪胎表面變得粗糙，從而提升抓地力。老美那邊的直線加速賽經常可以看到車輛在準備階段燒胎。）

-

轉速越低越省油

事實：勻速行駛時確實是這樣，但加速時就不同了：低轉深踩油門與高轉輕踩油門帶來的加速性能相同，但前者的耗油量比後者多，並且容易積碳。

-

想加速先升擋 / 超車要升擋 / 檔位越高車越快

事實：車輛動力輸出水平是由發動機轉速決定的，轉速越高動力越大。在車速固定時，檔位越低發動機轉速越高，輸出動力也就越大，所以想要急加速時先降一兩個檔位效果更好。

-

大街上轟油門的都是裝 X / 傻 X

事實：雖然被排氣聲嚇到確實很惱火，但是部分性能車啓動時必須轟一腳油門，或者排氣系統本身調教的聲音比較大、溫柔駕駛都聽起來像轟油門，還是希望大家能互相理解一下。（我有個騎本田二衝的朋友，每次到小區門口都是下車熄火推回車庫）

專門改排氣炸街的確實該罵。

-

單邊過減速帶會使懸掛變形 / 高速通過減速帶會損傷懸掛

事實：雖然這個謠言的初衷是爲了讓大家安全駕駛，但是懸掛系統沒有這麼脆弱，大家也不用在不小心軋過減速帶後心疼懸掛。相比於懸掛，高速通過障礙時更容易受傷的是輪胎與輪輞，如果輪胎都沒事，基本上懸掛也不會有事。（公司老闆的 C300 曾經有次高速軋過一個沒蓋的下水道，輪輞都幹開花了，懸掛一點事沒有）

-

對於買菜車來說，前驅後驅、前後重量配比並無意義

事實：你去超市買菜，推着購物車和拉着購物車感受是不同的，把菜堆在車頭和放在中間感受又是不同的。車重十幾千克、平均時速個位數的購物車尚且這樣，更不用說車重一噸多、平均時速三四十的買菜車了。

（不過目前對大多數人來說是這樣的：不管這個購物車好不好推，只要它能幫我把菜運到收銀臺就 OK。這樣想的話，這一條或許不算錯誤。）

-

多連桿一定比扭力樑好

事實：多連桿和扭力樑各有利弊，具體優劣要看調校與路況，不少賽車、運動型車都採用了扭力樑懸掛，比較有名的是歐寶家的瓦特連桿。

-

保險槓是爲了減少撞擊時車身受到的傷害 / 保險槓越結實越好 / 保險槓填泡沫是偷工減料

事實：保險槓是爲了減少你撞到行人時行人受到的傷害，同時在低速碰撞時提醒駕駛者剎車，避免傷到車身結構。過硬的保險槓會喪失這兩項功能。

-

SUV 安全性比轎車好

事實：雖然從各大統計數據看來轎車在事故中的死亡率比 SUV 高，但是這種統計都沒有做到控制變量——車身較高、車重較大的 SUV 在與轎車發生撞擊時有不可忽視的先天優勢。如果哪一天大街上跑的全都是 SUV 了，事故死亡率會不會不降反升還很難說。

-

車身越硬安全性越好 / 日系薄皮吸能死的快

事實：容易引戰，不過多討論

想了想還是說個題外話：

優質鋼化膜會在屏幕受到撞擊的時候碎裂，從而吸收撞擊能量，保證屏幕不受傷害。差的鋼化膜賊硬，受到撞擊時候自己沒事，能量都傳遞到內屏上，GG。

有人說鋼化膜太脆一磕就碎也是質量不好，不如造硬點好讓自己不用老換鋼化膜。我想問一句：屏幕貴還是鋼化膜貴？
