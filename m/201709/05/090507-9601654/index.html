在非洲，國家隊每贏一場球，留給大家打架的時間就不多了

足球告捷與民族團結——來自非洲的證據

戰勝烏茲別克斯坦之後，男足還保有一絲打入世界盃的可能。即使可能性確實只有一絲——既要自己爭氣，也要同行配合，球迷仍在期待。足球告捷，舉國歡騰，不僅中國如此，飽受族羣衝突之苦的非洲也如此。Depetris-Chauvin 和 Durante 的工作論文發現：國家隊在非洲杯或世界盃獲勝後，非洲各國民衆的國家認同增強，信任程度上升。此外，如果國家隊成功闖進非洲杯決賽圈，之後該國境內發生的暴力事件會減少。

數據來自非洲晴雨表（Afrobarometer）對 24 個非洲國家的調查。由於樣本很多，機構一般需要幾個月才能完成調查。如果中間發生了重要足球賽事，通過比較賽前賽後受訪者的回答，就可以估計足球賽的影響。研究涉及以下幾個問題：你更認同自己是 x 國人還是 x 族人？你是否信任你的親屬 / 鄰居 / 熟人 / 陌生人？你是否信任本族人 / 外族人？你是否支持現在執政的政黨 / 首腦？你覺得現在 / 未來自己的生活條件 / 國家的經濟狀況怎麼樣？

圖 1 獲勝比賽前後一個月時間，受訪者更認同自己是 x 族人的比例

結果：以獲勝前 3 天內爲基準，獲勝後 3 天內，“認同自己是 x 族人“的比例下降了4.4%，比起基準下降了20%！國家認同增強一直持續到 15 天后。對信任程度的分析得到了同樣的結果：無論是對本族人的信任，還是對外族人的信任，還是將親屬 / 鄰居 / 熟人 / 陌生人加總得到信任指數，獲勝都有顯著的正面作用。不過，勝利不會顯著影響人們對執政黨 / 在位首腦的支持程度，也不會顯著影響人們對生活條件 / 國家經濟狀況的看法。

文章做了很多檢驗：首先，比賽前後調查者 / 受訪者是否在性別 / 年齡 / 就業狀況 / 宗教信仰 / 族羣身份 / 語言 / 居住地方面不存在明顯區別；其次，友誼賽的結果沒有顯著影響；再次，無論是雙方在歷史上交手次數更多，還是淨勝球數更大，正面影響都會更大。不過，賽事在主場還是客場舉行影響不明顯；最後，僅僅是比賽進球更多沒有顯著的正面影響。此外，無論男女、老幼、教育程度，獲勝對國家認同和信任的正面影響都顯著。

圖 2 相比沒有入圍的國家，入圍國家發生暴力事件的數量的變化

最後，文章考慮了是否入圍非洲杯決賽圈對暴力的影響，方法是雙重差分：考慮那些直到預選賽最後一個比賽日前，仍然存在入圍可能性的隊伍，然後比較其中最終入圍和沒有入圍的兩類隊伍，接下來發生暴力事件數量的變化。結果發現：比起“失之交臂”的國家，“搭上末班車”的國家接下來 4 周內，發生暴力事件的數量 / 族羣間暴力事件數量 / 暴力事件傷亡人數顯著下降。總之，不論足球是否真的“高於生死”，足球絕不止“無關生死”。

參考文獻：Depetris-Chauvin, E., & Durante, R. (2017). One team, one nation: Football, ethnic identity, and conflict in Africa.
