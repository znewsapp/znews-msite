天涼了，秋天穿衣搭配，該備的基礎款心裏有數了嗎？

女生需要的基本款秋裝有哪些？

最近老是被催着寫基礎款秋裝~可我在南方還在稀里嘩啦地淌汗，和空調相依爲命哩~出門也是怎麼清涼怎麼來。請問秋天是什麼？連個影都沒見到！

不過貌似北方已經很秋天了。嗯嗯，還是有了這篇基礎款秋裝的攻略，保證全面又貼心，解決你換季買新衣的大煩惱。

一、上衣

  • T 恤

我一直覺得，春夏秋季節裏，買 T 恤就好像逛菜市場或超市一樣，一打一打地買；扔的時候也不心疼啊，穿壞了就淘汰掉，難道還指望着它今後能成爲歷史悠久的傳家寶麼？作爲居家外出的必備單品，T 恤必須得列在基礎款秋裝列表裏的 No.1。

黑、白、灰、條紋的，照着買準沒錯▼

不過，要說有點私心的話，上面這些顏色在我心中的分量的確不一樣。

我發現自己越來越愛條紋了~畢竟，黑白灰再怎麼經典，那也是純色嘛，看多了難免就會嫌太單調。相比之下，條紋 T 恤依然採用了經典元素，但顏色上豐富了一點點，卻也不至於太雜亂。

怎麼選條紋 T 恤，也是有講究的。市面上的確是有好多好多條紋衫，但，大多數都只是把條紋搬上了衣服，條紋衫本身的氣質，並不到位。

在我眼裏，最讓人感到特別賞心悅目的條紋衫有兩種：

一種是夏天裏的條紋短 T 恤，修身 + 短款 + 密集的細條紋。身材苗條的姑娘們穿上後很有少女的青春活力感耶~不知道夏天裏有沒有這樣穿過呢？▼

另一種就帶有一丟丟法式風情，版型寬鬆 + 稀疏的條紋 + 一字領 / 小領口，而且好多都會在領口以下的地方留出一片空白。閒適慵懶又帶一點優雅和精緻~沒錯，就是這種味道▼

既然是在秋天，那條紋衫自然是得選第二種啦。挑的時候記住「寬鬆」「稀疏的條紋」「一字領」「小領口」這些關鍵字就對了~

  • 襯衫

如果說上衣裏哪一件單品最能在設計上玩出新花樣，毫無疑問，答案一定是襯衫。

尤其是經過了一個夏天，看遍了各路時裝精們凹的造型，實在忍不住感慨呀：設計師們的想法都太巧妙，能在一件普通襯衫的基礎上把設計玩轉得新意十足▼

但，既然這篇說的是基礎款，咱還是先別急着放飛自我了。這些年裏穿過不少襯衫，但最後還是發現，穿得最多的、最最基礎的，也就三種▼

  • 棉質修身襯衫

是不是會留意到一些職場裝裏的襯衫都特別挺括幹練？沒錯，這裏說的就是這種。

注意，這一類的棉質並不是很柔軟、很鬆垮的，而是有一些硬挺和彈性。這種面料做出來的襯衫，穿上身後倍兒精神，很適合上班穿，風風火火，帥氣幹練▼

  • 牛仔 / 牛津紡襯衫

牛仔和牛津紡都不會硬挺有型，而是柔軟、舒適，很適合秋高氣爽的時候外出郊遊有沒有！▼

  • 雪紡 / 絲質襯衫

牛仔和牛津紡雖說很柔軟舒服，但是不會有垂墜飄逸的感覺。別擔心！想要這樣的 feel，交給雪紡 / 絲質襯衫就沒錯了，充滿了柔美溫婉可人的氣質，滿滿的女人味▼

而各種衣服單品，可以分爲四種不同的風格：

無明顯風格的、帥氣幹練風、休閒風、女人味。

剛好，上衣的這四件單品也對應了這四種呢，後文裏聊搭配時也會再提到：

無明顯風格：T 恤

帥氣幹練風：棉質修身襯衫

休閒風：牛仔 / 牛津紡襯衫

女人味：雪紡 / 絲質襯衫

（注：條紋 T 恤有一丟丟法式風格，但爲了分類方便還是歸在無明顯風格的 T 恤裏啦）

二、下裝

  • 褲子
  1. 無裝飾的牛仔褲

記得自從小學起，牛仔褲就是我春秋季節裏雷打不動的單品。這麼多年來，趕時髦穿過非主流的破洞和毛須邊牛仔褲，但最後才發現，這些不過是穿過一季就扔的玩意兒。真正貼心又百搭的，還得是一條簡簡單單的修身牛仔褲。深色的、淺色的、黑色的可以各來一條▼

無論別人怎麼反駁說它太普通，風格不夠強烈，都無法阻止它成爲衣櫥裏的常青款。

2. 九分西褲

每次被人問起上班穿什麼褲子，既幹練又不會太死板，我都會滔滔不絕地給人安利九分長的西褲。

只想說，這件單品的發明簡直太讚了吧！面料挺括，版型從上往下微微收窄，外加褲腿中間有一條燙跡線，利落不拖沓，幹練度滿分，穿去上班簡直完美~▼

就像日系通勤風裏的 OL 們，褲裝裏 99%都是九分西褲▼

要是覺得黑白灰的顏色太死板呢，大可以選擇豎條紋的，沒那麼單調▼

3. 闊腿褲

我可是闊腿褲不折不扣的鐵桿粉絲。有垂感的闊腿褲，秋天裏真是超好穿啊。因爲版型寬鬆，所以很休閒、很放鬆；但同時面料也是垂墜的，所以絲毫不邋遢，顯精神。

一直都對各種 T 臺上最 in 的潮流不太感冒，穿衣圖的就是舒適又好看。前一兩年闊腿褲火起來後，就一直是我的心頭好，不管今後潮流再怎麼變也會堅持穿下去~再提醒一下哦，挑闊腿褲的關鍵詞是「有垂感」，否則不小心挑錯的話，分分鐘就能感受到人生的艱辛啊！▼

  • 半裙

一說起裙子，可能有些姑娘就會囔囔：不都是裙子嘛，夏天穿的裙子直接延續到秋天穿不就可以了嘛？

唔……夏末秋初的時候，當然完全 ok 呀。但我還是喜歡把夏裙和秋裙區分一下，秋天穿的半裙，還是需要有點不同的：

面料和長度：

夏裙多半輕薄飄逸，還有不少膝蓋以上的。

但秋天裏還是需要穿得厚重一些來保暖；更何況，秋天裏身上的其它單品都會厚實一些，如果這時候唯獨裙子很清涼，這畫風也格格不入。

所以，秋天的裙子，面料需要更厚重，更挺括，更長▼

版型：

有沒有發現，秋冬季節裏，絕大多數半裙都是傘裙的版型，而直筒裙和包臀裙比較少見？

大概是因爲，秋冬季裏上半身都會穿得厚一些，所以下半身能體積大一些去平衡一下是最好啦。相比起其它貼身的裙型，傘裙自然就是最好的選擇了。

而且最簡單的純色傘裙還真的是怎麼搭都不不錯的好單品。

夏末秋初的時候，想要抓住夏天的尾巴，可以試試【傘裙 + 露腰短上衣】，比起夏天裏常見的【短褲 + 露腰短上衣】內斂了不少，卻依舊青春無敵▼

【傘裙 + 襯衫】甜美又斯文▼

總之呢，秋天裏的基礎款下裝，三條褲子加一條裙子就足夠啦：

無明顯風格：無裝飾的牛仔褲

帥氣幹練風：九分西褲

休閒風：闊腿褲

女人味：傘裙（半身）

三、外套

  • 長開衫

過去，我一直都喜歡的是短款的開衫，總想着，哎，反正開衫的禦寒功能也只有一丁點啦，長開衫穿着也未必會更暖和，乾脆就別那麼累贅，穿短款吧！

後來才發現，長開衫最重要的作用還在於形成縱向線條好麼！想要拉長身形？想要顯瘦？敞開的衣襟形成的縱向線條通通都幫你搞定~▼

  • 牛仔外套

雖說牛仔外套不如牛仔褲那樣足夠經典，但一說起休閒的穿衣風格，一定少不了它。

的確，很多品牌也在牛仔外套上玩出了特別多的花樣，比如加上貼布印花，或者讓它破個洞▼

時髦的樣子貨偶爾買買就好。但無論如何，一件剪裁修身、硬朗挺括、不帶多餘裝飾的牛仔外套，絕對是衣櫥必備▼

  • 小西裝

穿西裝就一定會穿成手拿保溫杯的中年老幹部？我看可不一定，至少下面這兩種小西裝，都能讓你穿成青春美少女呢！

  1. 休閒風小西裝

版型寬鬆、軟塌，面料柔軟，以棉麻爲主。其實這一類西裝裏，幾乎都沒有了西裝文化裏的嚴肅正式，風格還是偏休閒▼

2. 帥氣幹練風小西裝

版型修身，面料挺括，穿上身顯精神。沒錯，它妥妥的就是職場必備單品。對於上班族而言，黑白灰藏藍各來一件都不嫌多的~▼

  • 風衣

說風衣是衣櫥裏的經典款單品，想必沒人會否認吧？可能有姑娘會覺得傳統風衣太一本正經了，總想着挑一些花裏胡哨的款式。

可千萬別！那些漂亮的風衣圖，沒有一張是花哨款，但依舊可以穿出高顏值▼

所以，挑風衣時，還是得堅守經典款一百年不動搖。那些穿得時髦的，也就是在穿衣小細節上多花了些心思，這裏隨便提兩點~

比如，不要扣上釦子，而是把腰帶系一個結，這樣風衣的衣襟就能形成一個“V”字▼

繫好結後，記住結點不能位於身體的正中間，而是要移動一下，到側面一點▼

當然，這裏只是順帶提一提，今後如果專門聊風衣，還會再仔細聊穿風衣的那些小細節啦~~

長開衫、牛仔外套、小西裝、風衣，秋天裏的外套，全都齊活了！不僅兼顧了溫度高低，還 cover 到了不同風格哩：

無明顯風格：無

帥氣幹練風：小西裝（版型修身、面料挺括）/ 風衣

休閒風：長開衫 / 牛仔外套 / 小西裝（版型寬鬆、面料軟塌）

女人味：無

四、搭配方法

之前寫過一篇基礎款鞋子的文章（Sophia：女生需要哪些基礎款鞋子？），裏面也是按照風格把每一季的鞋子做了分類。

這樣，把秋天的上衣、下裝、外套和鞋子的按照風格羅列的話，就會有下面這張表▼

看過我之前基礎款系列的盆友們應該都很熟悉搭配方法了~

秋天裏，通常情況下全身都會穿 3-4 件單品。

套路 1：按照表格挑選，全部單品同一種風格

比如，大表姐身上的單品幾乎全是休閒風格的▼

套路 2：兩種風格

奚夢瑤的這身，和上面的大表姐相比，鞋子變成了帥氣風格的切爾西靴，是不是比單一的休閒風格多了些變化？▼

【休閒風牛仔外套 + 休閒風帆布鞋 + 女人味的傘裙】也是特別容易上手的一種穿法，既輕鬆悠閒，又充滿了十足的少女感▼

哈？至於全身單品 cover 到三種風格？呃，勸你還是別挑戰了吧，除非對自己衣品超自信。

連表格都給你們整理妥了，別告訴我還不會搭！！

五、進階款

如果你的衣品已經足夠好，上面的秋裝大法對你來說實在是小 case，那麼，再給你多推薦一件單品作爲進階款~~雖說不像上面那些單品那樣常見，但絕對時髦好穿。

它就是——吊帶裙。

要說剛剛這個夏天裏時尚博主們最關心的單品有哪些，slip dress 絕對榜上有名。

而且我總覺得吊帶裙自帶空調屬性啊，一穿上身就涼快百倍，比吃多少個冰鎮西瓜都還管用▼

只不過，夏天裏的吊帶裙多半都是絲質或雪紡的，入秋後就只能惋惜地暫時收進衣櫃，等着明年夏天再相見。

其實只要把吊帶裙的面料換成不那麼輕薄的，比如絲絨，依舊可以穿到秋天啊！

比起秋天裏最最常見的「上衣 + 下裝」，「吊帶裙 + 上衣」的穿法可以說是時髦很多了，並且很接地氣，一點也不浮誇▼

比如，【絲絨吊帶裙 + 白 T 恤 + 休閒風涼鞋】，減齡又少女▼

在這基礎上加上一件外套可以不？完全沒問題！要記住絲絨的面料本身就挺有女人味的，所以最好搭配帥氣幹練一些的外套喲，比如皮衣或小西裝，反差的感覺才迷人呢！

就像下面這身【絲絨吊帶裙 + 白 T 恤 + 皮衣 + 牛津鞋】，每一件單品拎出來看都再普通不過了，但湊在一塊兒就一派和諧的畫風啊！▼

提醒一下，挑絲絨的吊帶裙時，深色比淺色更不容易錯，畢竟絲絨還是很有膨脹感的，再加上淺色的話會顯胖吶。

嘻嘻，秋裝的所有攻略就全部在這了，get 到了啵？

最後忍不住把清單表格再放一遍，所有精髓全在這了！▼

在掏出錢包血拼之前，別忘了熟讀並背誦清單。這個秋天裏，想要快準狠地買買買，全靠它了！
