新車交付、股票大漲，Elon Musk 在躁鬱中走了一大步

一夜間讓特斯拉股價止跌大漲，馬斯克做了這些事

28 號的 Model 3 交付儀式上，人們看到一個搖滾明星般的 Elon Musk。

2 天后，Elon Musk 在推特上透露，自己可能得上了躁鬱症。

當時有人在推特上問：

我在 Instagram 上關注了 Elon Musk，看到他不可思議的生活。我很好奇，他的跌宕起伏，是否讓他擁有一個更過癮的人生。

Elon Musk 回覆說：

真相是，特棒的高潮、特糟糕的低谷和無情的壓力。別以爲人們想聽到後兩個。

緊接着，有人問馬斯克是不是得了躁鬱症，他回答，「是的」。

Elon Musk 解釋說，

也可能不是醫學上定義的躁鬱症，我也不知道，但壞的感覺總是和壞的事情聯繫在一起的。也許，真正的問題是，這是我自找的。

特斯拉 CEO 的這種狀態，和目前這家公司目前的狀況，關係很大。

特斯拉正處於一個至關重要的轉折點，邁過這一步，它就可以成爲在汽車市場上舉足輕重的大公司。否則，它就是一家產品小衆的小公司。

這一切取決於特斯拉首款價格適合大衆購買的電動車——Model 3——能否成功。

這也是爲什麼外界如此關注特斯拉第二季度財報和分析師電話會議的原因。

上半年的運營，是爲了 Model 3 放手一搏做最後階段的準備；後面的 6 個月，如果實現 Model 3 生產和服務支持體系的快速增長，對特斯拉未來至關重要。

從此以後，特斯拉進入了 Model 3 的時代。但它真的做好準備了嗎，Model 3 能成功嗎？這幾乎是每個特斯拉投資者的疑問。

儘管擔心自己得了躁鬱症，但 Elon Musk 再一次成功讓不少投資者們相信，Model 3 項目正處於正確的增長軌道上，特斯拉擁有光明的未來。

8 月 2 號美股開盤，特斯拉股價一度持續下跌，收盤時勉強漲了一點。但第二季度財報發佈和分析師電話會議之後，特斯拉盤後股價大漲 7%。

Elon Musk 是怎麼做到的呢？

從投資者（用戶）需求出發，擺事實擺數據，講道理講邏輯，這就是 Elon Musk 一次又一次讓投資者和用戶相信他的原因。

關鍵財務指標超出華爾街預期

這是投資者看好特斯拉的基本背景。這些數據在特斯拉財報裏都能很容易查到，可以去 http://ir.tesla.com/ 或雪球查看財報，我這裏就不貼出來了。

擔心下半年 Model 3 產能爬坡不能如期實現？你看，我們做了這些準備。

之前 Elon Musk 在 Model 3 交付儀式上一句我們即將進入「生產地獄」，嚇壞了很多投資者和分析師。

他們很擔心在隨後的兩個季度，特斯拉無法實現激進的產能爬坡計劃，導致 Model 3 銷量崩盤。

在第二季度財報和電話會議中，Elon Musk 用事實和數據，安撫了投資者和分析師。

特斯拉在第二季度的資本支出達 9.59 億美元，比第一季度進一步增加，主要就是在 Fremont 總裝廠和 1 號超級工廠裏，對 Model 3 的生產能力進行了大規模投資。

在特斯拉 Fremont 工廠，Model 3 的車身焊接線和多層通用裝配線都實現了高度密集、高度自動化。這些都讓 Model 3 快速、大量的產能爬坡成爲可能。

電話會議中，對 Model 3 的產能目標 Elon Musk 再次表現出樂觀態度。他表示有信心在今年年底實現 5000 輛 Model 3 的周產能，也就是 12 月特斯拉將生產出 2 萬輛 Model 3。

Elon Musk 和特斯拉 CFO 表示，還計劃在 2018 年的某個時間點，實現每週生產 1 萬輛 Model 3……

這是一個非常激進的目標。一旦實現，這將是汽車工業歷史上，最快的產能爬坡。

Elon Musk 說：

我說到生產地獄可是認真的。我們都很清楚即將遇到的困難，但我們買了門票，所以地獄也擋不住特斯拉。說實話，現在我對特斯拉的感覺相當好。

同時，特斯拉還投入資金擴建了銷售、售後的基礎設施。

第二季度，特斯拉新啓用了 29 家體驗中心和服務中心。再結合 Elon Musk 在交付儀式上承諾，2018 年超級充電樁數量是目前的 3 倍。緩解了外界對 Model 3 基礎設施的擔憂。

Model 3 賣得越多，虧得越多？那是謠傳

特斯拉說，2018 年開始大批量生產後，Model 3 能夠達到可與 Model S/X 媲美的毛利潤。

由於 Model 3 尚處於生產早期，會受到產能等因素較大影響。而第三季度，Model 3 的毛利率會受到人力成本等因素較大影響。

假設這些保持持平的情況下，隨着產能的持續擴張，預計第四季度 Model 3 能實現正毛利率。預計 2018 年 Model 3 的利潤率將達 25%。

同時，Elon Musk 還更新了 Model 3 最新訂單數據。

他在交付儀式上曾說，訂單已超 50 萬輛。電話會議中，他說之前的表達有誤。

總訂單量的確超過 50 萬輛，但已有 6.3 萬張訂單被取消，目前有效訂單還剩下 45.5 萬輛。關於這幾萬張訂單取消的原因，我會再寫篇文章分析。

Elon Musk 還補充說，交付儀式後，Model 3 每天新增訂單能達到 1800 輛。這依然是一個很高的預定量。

不過如果 Model 3 的產能爬坡出現問題，會引發更多的訂單被取消。

Model S/X 下半年銷量非常樂觀

之前高盛分析師發佈的一篇報告，曾讓特斯拉股價大跌。主要擔心就是 Model 3 一旦如期量產，將會侵蝕 Model S/X，特別是 Model S 的銷量。從而導致高端車型的銷量停滯不前。

特斯拉表示，高端車型 Model S/X 的訂單也在穩步提升。與 2017 年第二季度相比，7 月每週的淨訂單增長 15%。

特斯拉財報中寫道：

我們發現伴隨着 7 月 28 日 Model 3 的交付，Model S 的訂單也在進一步增加。來自消費者不斷增長的需求，讓我們有足夠的理由期待下半年 Model S/X 的交付量持續增長。

One more thing

Elon Musk 在電話會議中，連續 2 次打了自己的臉。

第一次打臉，就是推翻了自己之前宣揚的 Model Y 將採用全新平臺。當初 Elon Musk 理直氣壯的說，自己反思過了，SUV 就該有 SUV 自己的平臺，而不是用轎車的平臺修修改改。

所以 Model Y 不會和 Model 3 共平臺，而是採用更先進的平臺，啓用鉛酸電池系統，整個平臺將使用極少的線束，從而實現高度自動化的生產。

昨天，Elon Musk 說：

幸運的是，我的團隊把我從瘋狂的邊緣拉了回來。Model Y 將大量使用 Model 3 的平臺技術，更快進行量產。

第二次打臉，是他之前曾信心滿滿的表示將在 2017 年年內完成穿越美國的完全自動駕駛演示。Elon Musk 表示這個自動駕駛演示，估計要延期到 2018 年年初了。

我覺得這兩次打臉，表現出 Elon Musk 正變得更成熟。我相信他更想要一個全新平臺的 Model Y，但顯然這在工程上難度極大。不僅會影響 Model Y 的交付時間，公司財務狀況，更會嚴重影響用戶、投資者對特斯拉未來發展的信心。

在團隊的勸說下，Elon Musk 聽從了更合理的計劃，而且及時向公衆透露了最新計劃。

而推遲穿越美國的完全自動駕駛演示，顯然是因爲技術上還沒有做好準備，Elon Musk 也願意承認這些不足，而不是爲了面子堅持要啓動項目。

不過自從和 Mobileye 分手後，特斯拉在 Autopilot 方面的開發並不順利，項目負責人也不斷更換。這次演示推遲，也讓外界開始擔心 Autopilot 的開發進度。

希望 Elon Musk 和特斯拉，真的可以實現這些看似很難完成的目標。
