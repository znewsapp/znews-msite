特斯拉之所以成功，最重要的是做到了這一點

特斯拉爲什麼能成功

就在不久前，特斯拉的全新中型電動轎車 Model 3 已經下線。與 Model S 和 Model X 的高定價不同，Model 3 的起價僅爲 35000 美元，大抵與寶馬 3 系，奔馳 C 級處於同一水平。而因爲電動車在美國享受 7500 美元的聯邦退稅補貼，它在美國的實際售價僅爲 27500 美元，而在特斯拉的大本營加利福尼亞州，扣除州補貼後只有 25000 美元的售價，這一價格已經非常接近平價品牌中型轎車的售價。因此，Model 3 在公佈預售一週內，就收到了 32.5 萬份訂單，超過之前品牌最暢銷的車型 Model S 一倍以上。

可以說，今天的特斯拉已經取得了汽車市場的認可。在各大國際品牌價值榜單上，特斯拉品牌的估值都排在汽車品牌的前 15 名，遠遠超過了同樣來自美國的凱迪拉克和林肯，儘管後兩者成立的時間都已超過 100 年，而特斯拉僅僅在 2003 年才成立，比中國的絕大部分自主品牌都還要晚得多。那麼，特斯拉爲什麼能夠如此快速地在如此強調積累的汽車行業取得突破呢？

可能很多人認爲，這是因爲特斯拉造的是電動車。而 21 世紀電動車必將取代傳統燃油車，特斯拉作爲電動車領域的帶頭大哥，只是非常聰明得趕上了這一趨勢。其實，且不論電動車在現階段在很多方面還取代不了燃油車，特斯拉在電動車領域，也未必就是無可爭議的領頭羊。其實汽車剛誕生時，就有電動車，而在最近的一波電動車熱潮中，其實通用等很多企業在 90 年代就已經在電動車積累了很多技術經驗，可爲什麼傳統車企生產的電動車卻從未獲得類似的成功呢？

依我個人看法，特斯拉之所以成功，最重要的是對電動車進行了全新的市場定位。

過去傳統車企研發電動車，往往是受到政策的影響，主打經濟、節能、環保。但實際上對於現階段的電動汽車來說，儘管看起來耗電比燒汽油便宜，行駛中也沒有排放，但如果電力的來源不夠清潔，電動車也未必比燃油車更環保。而因爲儲存電能的電池技術仍然比較落後，要承載相同的能量，電池的體積、重量、成本都要比汽油高得多，而使用壽命和可靠性也不及燃油動力系統，因此即使扣除了政府的高額補貼，很多所謂的“經濟型電動車”也並不經濟，而且因爲電池佔據大量空間，往往車輛內部可用空間較少，而且因爲充電速度慢，充電設施也較少，使用起來也十分不便。因此，這些需要在“實用性”上與燃油車競爭的電動車難以成功也是很自然的了。

但特斯拉造電動車，從一開始就是“豪華”和“個性”的定位。電動車使用電機驅動汽車，除了效率高以外，其實駕駛體驗也有很多優勢。電機的運轉幾乎沒有抖動和噪音，也一般不需要變速箱，因此格外平穩、安靜、平順，而這正是豪華車需要的特性。電動機在低轉速就能爆發出最大扭矩，而且沒有內燃機搭配變速箱的各種遲滯，動力響應極好，提速超車格外隨心所欲。而且因爲電動機的成本和體積相對小，也相對容易在一輛車內堆砌更高的動力。在城市中行駛，特斯拉的動力感受幾乎是無敵的，加速時也仍然安靜、平穩的感受也極爲特別而有科技感。

這些動態方面的優勢配合以特斯拉科技感十足的內飾設計，使得特斯拉提供的用戶體驗足夠特別，使得它的定價能夠高到覆蓋電動車的高成本。而與此同時電動車其他一些實用性上的缺點，以及特斯拉因爲缺乏積累在其他方面檔次感的欠缺，對於購買這個價位車型的用戶來說反而不是問題——就像購買保時捷跑車的人，並不會介意它發動機聲音大，內部空間小而且長途行駛不夠舒適一樣。

從另一個角度來說，特斯拉通過電動科技打造出的品牌形象，正好與美國西岸的硅谷和西雅圖的年輕科技新貴的形象是一致。而這也是美國目前最拿的出手的形象，比林肯大陸代表的東岸 old money，和凱迪拉克凱雷德代表的南方陽光帶的洛杉磯和邁阿密等城市的暴發戶形象，在國際上都要有吸引力得多。

林肯大陸

凱雷德

Model S

不過，Model 3 的面世，對於特斯拉來說是一個重要的轉折點。過去特斯拉 Roadster、Model S 和 Model X 分別對應保時捷 911，Panmera 和卡宴，而二者的品牌定位也同樣是 luxury 豪華品牌。而 Model 3 的官方定價則已經完全向低一級的 premium 高檔品牌看齊了。看起來，特斯拉並不想成爲下一個保時捷或瑪莎拉蒂，而是想成爲汽車界的蘋果，做一個讓普通人買得起也願意多花錢買的大衆高檔品牌。如果這一策略得以成功，特斯拉贏得的將不僅僅是單車利潤，而是一個電動車生態的平臺。至於結果如何，就讓我們拭目以待。
