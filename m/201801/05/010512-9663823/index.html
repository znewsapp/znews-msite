大誤 · 怎麼寫出那種日式中二病的歌？

如何創作日式中二病風歌詞？

爲大家推薦一首集合了各類日式中二手法的歌。掌握這首歌詞，你也能寫中二病入股的日語歌詞。

可以點開鏈接一邊聽一邊看：空想メソロギヰ - 妖精帝國 - 單曲

《未來日記》OP《空想メソロギヰ（空想神話）》
專輯： 空想メソロギヰ
歌手：妖精帝國

手法 1：自譯片假名詞

標題是全文基調，必須中二！神話不能簡單地叫做「神話しんわ」，必須用英文轉出地片假名，片假名也不能直接用詞典上「Mythologyミソロジー」，非得自己翻譯一個「メソロギヰ」。召喚出古代假名「ヰ」才肯罷休。

Consentesdei/Juno/Juppiter/Minerva/Apollo/Mars/Ceres Mercurius/Diana/Bacchus/Uto/Vesta/Venus

手法 2：堆疊神話（傳說、天文、物理）意象

希臘神話、聖書、北歐神話、日本神話，稍微查了一下就扯到那邊去。內容不說明白聽都根本聽不懂。要告訴聽衆就好好唱啊。

這段歌詞變音很嚴重，對着歌詞都聽不明白，中二極了。

微睡(まどろ)みの淵で目蓋に揺蕩う // 稍微小睡的深淵中眼瞼在晃動
夢想の神は優雅に微笑み死を降し // 空想之神優雅微笑 散播死亡
時空を総べる因果の鎖は絡み出す // 統轄時空的因果之鏈交織纏繞
神の與えし空想 Program // 神所賜予的空想計劃

手法 3：高漢字比例（尤其是雙漢字）

這一段的漢字比例非常高，還是日常很少使用，用也很少寫成漢字的雙字詞。立刻脫離了人間趣味，浮上半空一個檔次。中二病需要的是和日常語言若即若離。

你會發現，漢字比例高，你還是不明白 ta 在表達什麼。

さぁ eins zwei drei! 重なり合う // 來吧 一 二 三 互相重合
さぁ eins zwei drei! 死を躱して // 來吧 一 二 三 迴避死亡

手法 4：上德語

英語已然不新鮮，畢竟已經是外來語的絕對龍頭。中二病也需要語言學上的創新，要用有檔次又小衆的語言。日本人似乎對德語的扮蟹效果特別中意。中文當中就不建議用德語來中二，似乎法語更適合。

消滅の遊戱に焦がれる奇跡を奪う Suvivalgame // 在滅亡的遊戲中奪取渴望的奇蹟 生存遊戲
螺旋のように // 宛如螺旋一般

手法 5：抽象名詞 +のように

不明就裏的比喻是製造奇幻感的重要手段，你看看這句話，什麼叫做“螺旋”一樣？比喻本來是把問題說得更清楚的，但是身爲一箇中二病，你必須把一件事比喻得更加抽象，才能增添更多的奇幻感覺。

Consentesdei/Juno/Juppiter/Minerva/Apollo/Mars/Ceres
Mercurius/Diana/Bacchus/Uto/Vesta/Venus
泡沫の未來書き換えるように // 爲了改寫幻如泡影的未來
空想繋ぐ輪廻のノイズが鳴り渡る // 維繫空想的輪迴的噪音響徹四周
時間は歪み因果の旋律は崩れ去る // 時間扭曲 因果之旋律邁向崩潰
神の與えし継承 Program // 神所賜予的繼承計劃

手法 6：佛系詞匯

日語當中的哲學詞彙，有很多是通過佛教傳進日本的。這部分詞彙自帶漢字加成和非日常加成，效果拔羣。事實上，這些詞彙，即使在中文中，也可以非常中二。

さぁ eins zwei drei! 連なり合う // 來吧 一 二 三 互相牽連
さぁ eins zwei drei! 死を振り切って // 來吧 一 二 三 拂開死亡
消滅の遊戱に抗う奇跡に挑む Erasergame // 在滅亡的遊戲中挑戰反抗的奇蹟 抹殺遊戲
夢幻のように // 仿如夢幻一般

手法 7：自制英文詞彙

外來語變多了，單純的外來語就滿足不了羣衆的需求了，就輪到和制英語上場了。其用法和英文並不一致。你看一下這個 Erasergame，什麼意思？橡皮檫遊戲？看不懂？看不懂就對了，這部動畫是一個大逃殺故事，於是就莫名其妙地用 eraser 表示“消除抹殺”，再組合一個“game”，自制了一個英國人絕對看不懂的英文單詞。夠不夠中二？

神の與えし機械(からくり) Program // 神所賜予的傀儡計劃

手法 8：隨意改動漢字詞讀音（寫作 XXX，讀作 XXX）

這一手法在這篇歌詞中不算是最典型的，但是各位浸淫日語 ACG 歌一段時間後，都會發現這種情況比比皆是，屢見不鮮。寫作「時間」讀作「とき」司空見慣，「とある科學の超電磁砲(レールガン)」更是經典之作。

さぁ eins zwei drei! 重なり合う // 來吧 一 二 三 互相重合
さぁ eins zwei drei! 死を躱して // 來吧 一 二 三 迴避死亡
eins zwei drei! 連なり合う // 一 二 三 互相牽連
さぁ eins zwei drei! 死を振り切って // 來吧 一 二 三 拂開死亡

你看這段德文已經用上癮了。

事實上，日語複合詞（比如：重なり合う）對於中文母語者來說也有一定的神祕中二效果，既有漢字，又看不懂，還滿滿的東洋異域風情，最後查一下字典，居然是很普通的詞組。

好了，我就不多說了，下面是最後一段歌詞，大家一起賞析一下，這日本式的中二風格吧！

消滅の遊戱に焦がれる奇跡を奪う Suvivalgame // 在滅亡的遊戲中奪取渴望的奇蹟 生存遊戲
螺旋のように // 宛如螺旋一般
哀憐(あいれん)なる傍観者 // 哀憐可悲的旁觀者
焦がれる生死よりも遠く // 將生死置之度外
最果てにある箱庭で // 最邊遠的盆景裏
終わりのない奇跡があると // 存在着永無休止的奇蹟
信じていた // 我深信不疑

以上。
