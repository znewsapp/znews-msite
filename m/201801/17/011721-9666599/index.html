《黑客帝國》裏的錫安是不是虛擬世界？

《Matrix》不是開放結局。Zion 是徹底的虛擬世界，而且到處都有明確的佐證。

先簡單講一下 Matrix 的整個架構。

背景：AI 誕生，最終與創造 AI 的人類發生衝突並取得了徹底的勝利。同時由於人類破壞了地球的大氣層，使得太陽這個地球能源的唯一來源被切斷，而電磁雲層機器又無法穿透，因此機器想到了用人類做能源來維持機器的運轉。（這裏大家就不要考慮能量守恆的硬 BUG 了）。

用人類做能源的方式類似“電池”，把人類放在一個培養皿裏（子宮），吸取他釋放的生物電。這一場景在電影中已經有充分的演繹，這也是 Matrix 第一集最震撼的場景之一。但是很快機器發現，沒有大腦活動的純電池，人類會大批死亡。因此，Matrix 第一代建立了。機器通過一根控制柱接管了大腦的輸入輸出系統，製造了一個虛擬的世界，讓大家在裏面虛擬地生活，以保持較高的存活率來供電。這時的 Matrix 很好理解，就是一個好比“魔獸世界”的網遊，把所有人類連進去，每個人賦予一個形象和定位在其中生活。

逐漸機器發現，總有 1%的人類，不管你怎麼賦予他們設定好的意識，他們都會發現不對勁的地方，都會產生自我意識的覺醒。對於這些人，系統又設定了第二個虛擬世界，Zion，大家可以理解爲另一個網遊。只不過這個網遊的設定是基於真實世界的。這個世界每隔一代 Neo 都會被毀滅一次，實際上 Zion 是一個“內測服務器”，目的是爲了讓 Matrix 更完美而存在的。

因此整個 Matrix 電影的架空世界就很好理解了：

主網遊：Matrix，玩家：全人類 99.99%的人口

測試網遊：Zion，玩家：約 0.01%的人口

真實世界：跟 Zion 的設定一樣，但沒有野生人類存在

Matrix 是網遊大家都能理解。真實世界的狀況大家也都認可。但 Zion 也是一個網遊，影片中沒有直接演出來，因此還是有很大爭議。但是導演已經給出了足夠多的蛛絲馬跡，告訴我們這個事實。

“Zion 是虛擬的”的證據：

Neo 的超能力在 Zion 中生效。

Neo 其實包含兩部分，他作爲人的自我意識，以及機器在他的意識裏埋下的代碼（Code）。這部分代碼有很高的優先級和系統權限，因此在他覺醒後，可以在 Matrix 裏飛行和做到各種超人才能做得事情。而系統權限則是他可以看到綠色代碼構成的 Matrix，意即看到了 Matrix 的本質。

可是我們發現，Neo 的超能力甚至帶到了 Zion 中。由於救世主的權限很高，他甚至在 Zion 網遊中也具有一定的權限。甚至在他被 Smith 弄瞎後，看到了金色代碼構成的 Zion。其實這時 Neo 已經悟到了，纔有了後面和機器之神的對話。金色的代碼啊各位，區分於 Matrix 的綠色，導演的用意還不明顯嗎？不要用什麼腦電波，意識流這種軟的不能再軟的科幻來解釋這一切了好嘛？單手滅烏賊也好，金色代碼也好，已經證明了 Zion 這個測試服務器的本質。

Smith 的 Zion 入侵

Matrix 系列裏，機器是有意識的個體。這些個體有着明顯的自我意識，因此有的各司其責，有的也會害怕被刪除而藏起來。甚至還有火車站這種“私服”用於跟 Matrix 服務器交換數據。Smith 也是這些機器意識中的一員。他被 Neo 幹翻之後，意外地獲得了一些系統權限，因此他變成了一個病毒。在影片的最後，這個病毒已經幾乎複製了整個 Matrix，甚至危害到了整個 Matrix 的穩定性。這裏不展開講。

Smith 作爲一個機器，竟然能複製進 Zion 中的一個個體中，這裏如果用意識流，腦電波這類軟科幻來解釋，真的好嗎？

讓我們回顧電影中插在腦後那根插頭的定義：那只是接管了大腦的輸入輸出系統啊！憑啥可以讓機器整個植入到人的意識中且可以斷開後獨立作業？即使救世主 Neo，帶着救世主的 Code，也是因爲 Zion 還是虛擬的，這一切才存在；如果真的把他的插頭拔了，哪兒還有什麼救世主的 Code 攜帶呢？Smith 複製進 Zion，又是一個 Zion 虛擬的鐵證。

其它旁證

以下不是鐵證，但也都說明了 Zion 是虛擬的這一現實：

機器烏賊嗚秧嗚秧，越打越多；

設計師和 Neo 的對話：原來 Zion 已經毀滅過 6 次。請問如果是現實，如何毀滅 6 次？現實中重建一次 Zion 和程序重啓一下難度差距也太大了吧！在能源如此匱乏的 Matrix 架空世界裏，這樣做是否太奢侈？烏賊也是同理。

其實這部電影最精彩的部分並非虛擬與現實，而是自由意志（Free will）的哲學思辨。人到底有沒有選擇，是否有自由意志，一切是否是註定的？Oracle 爲何可以預測很多事情？這纔是影片最精彩的核心內容。

首先提出我的觀點，ZION 是虛擬的。

ZION 之所以存在，是矩陣不完善的緣故，導致其中百分之一的人類能察覺到，所以矩陣設計師就重新建了一個虛擬世界來安置這一部分人，否則矩陣會崩潰！而矩陣崩潰的話裏面的人會死，隨後整個機器世界因爲缺乏能源也會完蛋，所以機器世界設計師需要一次又一次來完善矩陣來達到人類和機器共存的目的，這是《黑客帝國》這部電影的真正主線。而 NEO 實際上是一個程序，一個用來改造升級矩陣的程序。

有證據嗎？有的，機器世界的矩陣設計師在召見 NEO 時，提到以前曾經設計過兩個矩陣，但都失敗了。所以矩陣設計師就造就了 NEO 這個程序，讓他和人類互動，從而更好地瞭解人類以及人類社會，汲取人類的感情與思想來使矩陣不斷地進化。

那 NEO 爲什麼身爲程序，卻站在了人類的對立面？這就是他被設計出來的目的！那一個程序怎麼擁有人類的感情？這不奇怪，矩陣中那個印度工程師是電源程序，但他對他女兒的愛顯而易見，火車人也是程序，但他明顯對 NEO 有着恐懼，可見隨着機器世界的進化，機器世界也慢慢進化出了感情。

所以 NEO 領悟到自己是救世主爲什麼在第一層矩陣中立馬就勢不可擋？那是因爲他覺醒了他自己的能力——他是一個具備特殊能力的程序，所以能看穿了這個世界只是代碼而已，所以這個世界中的所有物體對他來說只是普通程序，以他的權限完全可以摧毀！哪怕史密斯他都不怕，至於史密斯爲什麼也有特殊能力以及後來多了複製能力，且按下不表。

根據劇情，尼奧已經是第 N 個了，而且設計師對他的對話來看設計師對 NEO 瞭如指掌，這是 NEO 是程序的又一個印證！設計師讓他做的同樣決定，前面的 N 個救世主都按照機器一樣的理性來做了，從而前面的矩陣一直都停滯着沒有大變化，但 NEO 比起他的前任多了“愛”這個東西，所以 NEO 做出了與前人不同的決定，這是矩陣能夠升級的重要原因。

爲什麼 ZION 裏的人不知道他們在矩陣中呢？矩陣設計師預先有措施了，比如，NEO 在第二部認識到了 ZION 只不過是另一層矩陣，就輕鬆滅掉烏賊，這超越了他本身的權限，所以觸發了系統的自動保護，尼奧被系統回收了。個人猜想這就是矩陣設計師的預防措施。

矩陣升級是殘酷的，矩陣設計師在給 NEO 選擇的時候已經透露之前滅過五次 ZION 了！可見死掉了相當一部分人類! 那指點人類反抗的先知又是什麼呢？ 按設計師的話說，先知是矩陣的母親，所以史密斯作爲矩陣的一部分也會尊稱她爲母親，她對整個矩陣都洞若觀火。不得不說，先知對人類是親善的，但就像是歷史上的改革派和保守派，對農民政策再好，也是爲了統治階層服務！她是機器世界的程序，所以她的目的也是通過讓機器與人類共存從而更好的升級矩陣，其實機器世界真真正正的終極目的並不是設計一個合理的矩陣讓人類供給能源，而是最終讓機器世界不停的在與人類對話中不停進化，建立機器的理想社會！ 《黑客帝國》這個思想絕對是革命性的，因爲這部影片拋棄了人類社會中心論，還記得第一部史密斯對墨菲斯說什麼了？“我們取代了你們，這叫作進化，就像恐龍滅亡一樣。你們現在像病毒一樣，只知道耗費資源，我們機器纔是解決這個問題的辦法。”
