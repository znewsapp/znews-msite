2017 年度盤點 · 我心目中的年度十佳電影

兩週過去，「2017 年度盤點」專題到這裏就要跟大家說再見啦。

感謝你一直讀到這最後一篇。回望剛剛過去的 2017，如果有某些你難忘的行業、事件和人物日報未能提及，或者你對某一篇回顧尤其有感觸，歡迎在評論區留下你的想法。

2018 年，日報期待着和各位一起成長。

電影是另一種無限貼近理想的生活。它可以靈動輕盈，也能深刻內斂，人類總在無限地貼近它，爲它着迷，爲它瘋狂。迄今爲止，個人 2017 年度佳作如下。

外語片十佳：

1.《尋夢環遊記》

導演: 李·昂克里奇 / 阿德里安·莫利納

編劇: 李·昂克里奇 / 阿德里安·莫利納 / 馬修·奧爾德里奇 / 詹森·卡茨

主演: 安東尼·岡薩雷斯 / 蓋爾·加西亞·貝納爾 / 本傑明·布拉特

意料之中關於夢想、家庭、親情的故事，情理之外的陰謀與轉折。奇幻絢麗的亡靈城，噴涌而出的墨西哥元素。雙重死亡的設置，很具想象力。電影完成度很高，卻有着動畫難得的緊湊感，沒有浪廢一個多餘鏡頭。皮克斯總能精準戳中，人們那部分叫愛的軟肋。

2.《看不見的客人》

導演: 奧里奧爾·保羅

編劇: 奧里奧爾·保羅

主演: 馬里奧·卡薩斯 / 阿娜·瓦格納 / 何塞·科羅納多

多重反轉，沒有 bug，敘事流暢，剪輯上佳。電影后半段，導演奧里奧爾·保羅在已知結局的前提下，拍出了不一樣的懸念，有種東野圭吾小說的感覺。人性之惡，一覽無遺。從完美陌生人到看不見的客人，近年來的歐式小成本懸疑片，劇本簡直玩出花。

3. 《摔跤吧！爸爸》

導演: 涅提·蒂瓦里

編劇: 比於什·古普塔 / 施熱亞·簡

主演: 阿米爾·汗 / 法緹瑪·薩那·紗卡 / 桑亞·瑪荷塔

熱血感人的運動片，三傻之後又一部激起強烈共鳴的印度電影。印度的童婚比例極高，看似封建的爸爸辛格，實際給了女兒們追求夢想，尋求男女平等之路。同屬亞洲影壇，印度韓國電影都發展迅速，國內啥時能多些具有社會意義的好片？

4.《敦刻爾克》

導演: 克里斯托弗·諾蘭

編劇: 克里斯托弗·諾蘭

主演: 菲恩·懷特海德 / 湯姆·格林 - 卡尼 / 傑克·勞登 / 哈里·斯泰爾斯

驚懼的士兵，未知的歸途。昏暗的天空，陰雨的天氣。《敦刻爾克》並不是那種陳舊的戰爭片，實質上是一部懸疑片。沒有冗長的對白，也沒有浪費一個鏡頭。多重敘事，三線並進，懸疑色彩濃厚。簡潔的劇情，精煉的臺詞。沒有用綠幕進行拍攝，進行後期特效合成。而是採用實景拍攝的方式，視覺效果更爲真實。渺小如同螻蟻的士兵們，無處可逃的絕望。絕望到最後，變成希望。陰暗久了，終於看到光明。活着意味着一切，這亦是一場勝利。不說大人物的傳奇，只講普通小兵們的故事，卻是近期來最打動人的二戰片。

5.《天才槍手》

導演: 納塔吾·彭皮里亞

編劇: 漢塔維瓦塔娜 / 皮婭羅姆娜 / 彭皮里亞

主演: 茱蒂蒙·瓊查容蘇因 / 查儂·散頂騰古 / 依莎亞·賀蘇汪 / 披納若·蘇潘平佑

青春不只有戀愛籃球和考試，還有打架逃課抄卷子。節奏緊張，剪輯緊湊。男女主弱雞版王大陸 + 泰國杜鵑，演技顏值都在線。班克的內心活動 BGM：當初是你要作弊，作弊就作弊，現在卻想用真愛，把我哄回來。

6.《海邊的曼徹斯特》

導演: 肯尼思·洛納根

編劇: 肯尼思·洛納根

主演: 卡西·阿弗萊克 / 盧卡斯·赫奇斯 / 米歇爾·威廉姆斯

生容易，活容易，生活真不容易。卡西·阿弗萊克演技太贊，把這個喪到極點的男人，演繹的令人感動。好似黃昏的那道微光，張張慌慌，有些悲傷。

7.《請以你的名字呼喚我》

導演：盧卡·瓜達格尼諾

編劇：盧卡·瓜達格尼諾／詹姆斯·伊沃裏／沃爾特·法索

主演：提莫西·查拉梅／艾米·漢莫／邁克爾·斯圖巴

充滿青春和荷爾蒙氣息的愛情片，鏡頭底下八十年代意大利的夏天，恬靜而迷人。愛是超越性別，超越國籍的。只是你出現的那個時間剛剛好，於是來自靈魂深處的共鳴擊中了彼此。從猶豫、試探，到喜悅、迷離、靈肉合一，再到揮淚告別。這些愛中的小情感，把握的很到位。

8.《銀翼殺手 2049》

導演: 丹尼斯·維倫紐瓦

編劇: 漢普頓·範徹 / 邁克爾·格林 / 菲利普·迪克

主演: 瑞恩·高斯林 / 安娜·德·阿瑪斯 / 傑瑞德·萊託

身邊有好幾個人推薦過，但我還是要在這裏繼續安利。上半年看過《降臨》的話，應該對導演丹尼斯·維倫紐瓦並不陌生。科幻電影探討的主題多半鄭重、深刻、宏大，這部也不例外，科技發展、人性掙扎、正邪激戰，電影中充滿了符號化的鏡頭和充滿哲理的對話。導演把營造氛圍也推到了一個極致，畫面是不正常的色調，天氣不斷變換，科技感撲面而來，畫面美到驚豔，全片風格偏文藝。

9.《每分鐘 120 擊》

導演：羅賓·坎皮略

編劇：羅賓·坎皮略／Philippe Mangeot

主演：納威爾·佩雷茲·畢斯卡亞特／阿黛拉·哈內爾／阿諾德·瓦羅里斯

這可能是我今年最期待的一部同志電影，它的主題角度並不新穎，愛、病、死亡，都將被無限探討。但羅賓·坎皮略拍出了羣體和個體的交織，宏大與渺小的對比，他本來就擅長用詩意的畫面，沉穩地推動劇情，可以說才華橫溢，所以這次獲得戛納電影節評審團大獎、國際影評人費比西獎和酷兒棕櫚獎，我並不意外。

本身故事是極具爭議性的，它的切入點在上世紀九十年代，大家對艾滋病知之甚少，很多人誤解了艾滋病和同性戀者，在這個背景下，法國成立了一個叫 Act Up 的組織，主要向大衆科普知識，內森和西恩是組織成員，他們兩個個體與還處於矇昧的社會之間，摩擦出了憤怒、焦灼、絕望的火花。劇情遠不止這些，還有更深刻的呈現。另外，可以看到幾個主演的演技一直在線，實在不忍錯過。

10.《至暗時刻》

導演: 喬·賴特

編劇: 安東尼·麥卡滕

主演: 加里·奧德曼 / 克里斯汀·斯科特·托馬斯 / 本·門德爾森 / 莉莉·詹姆斯 / 斯蒂芬·迪蘭

敘事很贊，完全被表演吸引，不覺枯燥。加里·奧德曼演技值一個奧斯卡影帝，演出了多角度，形象豐滿的丘吉爾。酒鬼，脾氣暴躁，卻是整個大英帝國反納粹，最值得信賴的領導者。和《敦刻爾克》結合起來看，別有一番味道。不過英國骨子的勞資天下第一，最後的看門人，讓人有些不舒服，難怪老美老法呵呵噠。

特別推薦：《出租車司機》可謂韓國年度最佳。 《愛樂之城》我是 2016 年就看過了，所以放在 2016 年的年度片單裏。

國產片五佳

1.《二十二》

導演: 郭柯

類型: 紀錄片

理性的紀錄片，看似平淡鬆散，卻讓人在影院流淚好幾次。老人們的敘述回憶，沒有特別煽情，沒有特別凸顯強烈仇恨感，只是慢慢的講着那些過去。她們是被迫充當“慰安婦”的受害者，她們努力的活着。這部紀錄片最大的目的，不是引起民族仇恨。而是讓年輕人知道那段歷史，不要遺忘過去。

2.《嘉年華》

導演: 文晏

編劇: 文晏

主演: 文淇 / 周美君 / 史可 / 耿樂

不像《熔爐》那麼煽情，卻更加冷眼看待病態的社會現實。玩忽職守的警察，做假證的知名醫生，直面這些黑暗，能過審很不容易。凸顯出當代家庭教育問題，讓人深思不已。瑪麗蓮·夢露雕像是電影的重要線索。小米騎車逃離，與雕像擦肩而過也象徵了女性意識的覺醒和反抗，給了絕望世界裏的一抹亮色。

3.《相愛相親》

導演: 張艾嘉

編劇: 張艾嘉 / 遊曉穎

主演: 張艾嘉 / 田壯壯 / 郎月婷 / 宋寧峯

細膩而真實，感人卻不煽情。女兒與搖滾歌手的感情好像一杯可樂，清涼爽口，卻稍縱即逝。母親與父親的感情好像一杯白開水，看似平凡，卻不可或缺。姥姥的感情好像一杯白酒，辛辣嗆口，回味時卻別有感動。這三個女人形象，也是三個時代中國人剪影。

4.《大護法》

導演: 不思凡

編劇: 不思凡

主演: 小連殺 / 圖特哈蒙 / 金士傑 / 幽舞越山

質量強於《大魚海棠》，完成度高於《魁拔》。真·成年人向動畫，作者電影風格強烈，對於現實社會的映照，諷刺足夠辛辣。本片雖然在轉場和節奏感，處理稍有瑕疵，但整體觀感在水準之上。走的黑色幽默路線，有種昆汀版血腥暴力美學的味道。

5.《不成問題的問題》

導演: 梅峯

編劇: 梅峯 / 黃石 / 老舍(原著)

主演: 範偉 / 殷桃 / 張超 / 史依弘

有點像《驢得水》，借民國來諷刺如今的人情社會。有本事但爲人古板的尤大興，被處處打壓。左右逢源圓滑如泥鰍，卻沒有本事的丁務源竊居權位。坑蒙拐騙馬屁精秦秒齋，得到領導青睞。黑色幽默，忍俊不禁。三個男人一臺戲，活生生的大陸職場教學片。

無聲電影、有聲電影、黑白電影、彩色電影，科技在進步，電影的發展也在不斷革新。此後，無數人通過攝影、剪輯、燈光、故事等方式融入它，也有更多的人開始爲它着迷和瘋狂。電影已經存在了這麼久，它創造價值從未減少，反而因爲喜歡的人越來越多而不斷增加。電影這個造夢機器正在給更多普通人一場美夢。
