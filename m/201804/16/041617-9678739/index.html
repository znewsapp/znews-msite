憑什麼當了明星就能瘋狂圈錢？

明星爲什麼可以瘋狂圈錢？- 關於人格化偶像商業模式之週末思考

十年前，中國的知名女藝人流行嫁商人；而現在，中國的名藝人流行圈內內部消化，女星男星結合一下。

因爲名藝人實在太有錢了，有錢到靠自己一個人就比中國 99.999%的商人有錢的多，有錢到女性要找一個比自己有錢的男商人都有點困難。

對應着一現象的，是中國文娛影視產業近十年的瘋狂發展。從商業的層面，十年前，一部國產電影票房過 1 億就非常厲害了，而現在的國產電影票房可以過 50 億；十年前，一個綜藝節目拿到的廣告贊助是千萬級的，現在有節目能拿到 7 個億；一大批影視公司，旗下有兩三個名藝人，拍幾部劇，就能做出上億的淨利潤，就敢估值 10 億以上搞資本運作，甚至直接申請 IPO。而落到個人層面，十年前，國內知名藝人蔘演電視劇的片酬是按百萬來談，現在小鮮肉明星開口就是幾千萬。而對於一線明星，早就不滿足片酬，而是自己投資電影，甚至持有大量影視公司股權，通過資本市場來加速裂變。按照現在的形勢，如果目前有意向上市的一批文娛公司都成功 IPO，那麼中國身價 10 億以上的演員就有一大批，可以這麼說，只要你聽過名字，電視上看到經常露臉的藝人，沒有不豪車豪宅的。

面對這樣的情況，社會上自然有越來越多的非議。核心問題在於：“爲什麼戲子們都這麼有錢，比真正爲社會做貢獻的科學家、工程師、醫生等有錢的多，甚至比大部分企業家還要有錢？這公平嗎？”。確實，藝人的收入已經大大超越了普通人，甚至可以說，“貧窮限制了大家的想象力”。我之前沒有太在意這個情況，因爲對這個行業不熟。但最近看了一些文娛影視公司的財務情況，也是感到了震驚。我從事創業和投資十年，看過上千個企業，但從來沒有見過文娛影視公司這樣強悍的吸金能力。要知道，一個製造業、服務業的老闆，奮鬥 20 多年，僱傭了幾千上萬人，忙得頭禿了，肚子大了，一身慢性病，才逐漸登上人生巔峯，也就把公司做到每年幾千萬淨利潤，勉強去申請 IPO。即使是大家認爲暴利行業的房地產，絕大多數中小老闆們也是挺苦逼的，折騰了半天，錢都被政府和銀行拿走了，年淨利潤有幾千萬都不錯了。但文娛影視公司呢？這是完全的輕資產模式，手裏一個 20 多歲小鮮肉，賺得錢就抵得上你製造業上千員工，一個小鮮肉的收入可以是一千個工人的工資，太可怕。能和文娛影視的造富能力相提並論的，只有互聯網科技行業了。但互聯網起碼還是要有產品、技術、運營的，起碼還是個正經企業，得靠產品說話。而影視公司，在經營上要簡單暴力得多，有名藝人在就已經成功了一半。另外，如果考慮到大部分互聯網公司都是燒錢模式，而文娛行業是每個項目都要大賺現金的，且大部分收入落到名藝人個人身上，那麼我們且不提娛樂公司的商業模式好不好，名藝人本身的商業模式肯定是非常的好了。

平心而論，這個問題並不是中國特色。在美國同樣也是如此，名藝人的收入也一樣高到離譜。只是這個行業之前在中國比較落後，最近十年爆炸性發展起來了，大家才越來越關注到這個現象。那麼，問題來了，有沒有人認真想過，爲什麼名藝人不從事任何物質的生產製造，甚至不從事虛擬產品（一個網站，app 或者遊戲）的生產製造，靠刷臉就能刷成富豪？在網上看了一下，沒有答案。因此，我試着自己思考了一下，大概想法如下：

第一，藝人個人並不是不製造產品，她的產品是一個和她同名的人格化 IP。其實，陳道明已經在某牛奶廣告裏透露了這一奧義，“最難演的是自己”。對於藝人來說，世界必須是分裂的，因爲生活中的“我”，和作爲一個明星 IP 的“我”，是合一的，無法分割的，但事實上又確實完全不一樣的。前一個“我”，努力在打造後一個“我”，而後一個“我”，纔是真正賺錢的邏輯所在。你不能從“人”的角度來理解藝人，而是要從 IP 的角度來理解。人是個動物，但 IP 則完全是可以從商業角度來設定的，設定的初衷就帶有完整的商業目的，影視、戲劇、綜藝、廣告、代言，甚至自有品牌，一整個鏈條都已經設定好了。而且，這個 IP，不是藝人個人的貢獻，而是一整個團隊的努力結果，只是掛在藝人個人身上無法分離出來而已，這也是爲什麼藝人無法過正常的生活，沒有隱私的原因。藝人賺的錢，名義上是“她”賺的，其實是另外那個“她”賺的。

第二，作爲產品的人格化明星 IP，價值有時候甚至要大於作爲產品之產品的影視作品。確實，文娛作品也很賺錢，無論是電影、電視還是綜藝等，做的好現在都是大筆吸金。但是影視作品本身作爲項目制模式有三個問題。首先是高風險性，開拍的時候沒人知道是賺錢還虧錢；其次即使這一次賺錢了，下一次再拍還是沒人知道賺錢還是虧錢；最後，文娛作品的很大一部分投資，用來支付給名藝人了，或者說，支付了這個明星 IP 的“使用費”，明星 IP 本身會不斷吞嚥影視製作的利潤空間，這也是雖然文娛行業賺錢，但風險投資參與投資文娛影視製作公司並不多的原因。因爲長期來看，影視製作公司本身並不是一個好的商業模式。但反過來，藝人本身卻是一個好的商業模式，只要 IP 出名了，有了鐵粉，而藝人本身的臉就對應這個明星 IP，不管一部作品是否成功，這個 IP 具有持續性，因此商業化的吸金能力也在持續。基於這樣的情況，藝人相對製作方就強勢，獲得大量的收入也是可以理解的。現在來說，影視公司的應對手段有兩個，一是製造影視作品本身的 IP，即“系列”，希望這個故事 IP 來稀釋明星 IP 本身的影響，二是和名藝人妥協，通過分票房和分股權等方式，來降低固定成本，圈定名藝人。

第三，人類對於人格化明星 IP 的需求一直強烈，並不是現在纔有。人生本身在哲學層面上是荒誕的，也無法確認有什麼意義，人生的意義都是個體自己賦予的。當一個人有熱愛，有嚮往，有希望的時候，他會感覺幸福快樂，起碼，能緩解對人生本質的焦慮、困惑和憂鬱。因此，大多數人都是希望有感情投注的標的，而這個標的，最合適的是另外一個基於人型的 IP，而不是抽象的“道”和“理”，因爲這個世界的絕大多數人類，根本不具備抽象思維能力，而人類本身的思維範式也使得類人 IP 更容易讓人接受。人們感情投射的標的，常見的是父母、兒女、妻子或老公這樣的，但這些是身邊生活中的。如果要讓你產生更大的願景，明星 IP 的需求就應運而生了。在遠古，這就是神話人物。神話英雄的 IP 是人們感情投射的需求產生的結果，接收人們的喜愛和敬仰，被古人口口相傳。再舉個更容易理解的例子，就是宗教。每個宗教都有其中深刻的“道”，如果你有能力，其實可以直接去悟的，比如我個人喜歡的佛教禪宗。但是，事實上每個宗教都做了非常重的“擬人化”設定，比如佛祖，耶穌。爲什麼？因爲廣大羣衆喜歡這樣的人格化設定，他們有需求去崇拜一個高高在上的人，而不是崇拜一個透徹的“道”。如果你覺得當代的名藝人收入嚇人，但是和這些頂級的明星 IP 給自己體系帶來的收入，對人類歷史的影響比，簡直不值一提。

第四，基於人格化 IP 的粉絲運營在媒體渠道發展進化後進一步爆發力量，獲得粉絲將更多、更快。所有基於明星 IP 的商業模式，核心都在於粉絲運營。但是，在過去，基於信息渠道的不通暢，大規模粉絲運營面臨效率低，成本高的問題。比如潘安作爲一個帥哥，就不可能取得鹿晗吳亦凡的粉絲量，因爲只有他所在那個小城 的人能知道他，其他地方的人並不知道，沒有照片，沒有視頻，更沒有粉絲團交流，怎麼能感受到帥哥風采呢，感受不到。在古代，也只有幾大宗教做到了強力的粉絲運營，這是通過線下連鎖的方式，到處建網點，用了幾百上千年達成的。但是，隨着科技的發展，一切都不一樣了，因爲信息傳播方式發生了天翻地覆的變化。我認爲現代意義上的明星藝人，事實是在有電影技術以後產生的。電影技術使得記錄藝人活動的視頻拷貝可以跨越地域傳播，這是文娛歷史的一個轉折點。然後隨着電視的發明，傳播速度又上了一個臺階，老百姓可以在家裏實時地接收某個明星的視頻信息。如果你觀察美國的文娛行業，非常明顯地可以看到隨着電影和電視的普及，帶來兩波巨大的浪潮。當然，互聯網是第三波，是在電視基礎上的進一步加強，人們拿着手機，可以隨時隨地地看電影、電視、綜藝、廣告等，接收某個明星的信息。而讓我們換個角度，其實是技術手段讓明星 IP 這一方的粉絲運營效率越來越高。當初的宗教明星 IP，花了一百年才圈了十萬粉絲，而如今上個快手一晚上就可以圈十萬粉絲。這樣速度，正是快手大 V 月入數百萬的背後原因。粉絲運營強力程度高了萬倍，而粉絲們的智商，千年以來並沒有提高過。

第五，未來隨着物質供應的越來越過剩，人格化明星 IP 對粉絲收稅這一現象必然還會加強，單粉絲貢獻將更大。剛纔說到粉絲的智商千年以來沒有提高，這不完全是貶義。事實上，我覺得這不完全是智商的問題，而是涉及很深的東西，人性，人生的意義，以及生死。人類相比其他動物，太聰明瞭，我們一開始就知道自己來到這個世界就沒法活着回去，只能在這短暫的一生中避免痛苦，尋找一些意義，大部分人是沒有能力思考終極問題的。即使思考，除了帶來痛苦抑鬱，也不會有其他結果，因此他們都會選擇消磨時間，消磨時間的核心是娛樂，而娛樂的重點就是需要人格化 IP 的感情投射。人總有一種傾向，去暗戀，去喜歡，去崇拜，去認同，那麼一個神一樣的存在，無論是女神，還是男神。這個時候，明星 IP 的運營方就出現了，把各色完美的明星送到你面前，電影，電視，各大互聯網平臺，你不可能躲開，且總有一款你喜歡。在過去，其實人們要消磨的世界，並沒有那麼多，因爲大量時間要從事生產勞動，不然明天要沒飯吃，這個問題比追明星更要緊。但現在，時代變了，全球來說，物質生產已經極度豐裕，未來會更豐裕，人們不會再爲沒飯吃發愁，但大部分人又沒有機會真的改變自己的現實生活。那麼，人們一定會花更多的時間在娛樂上，在精神消費上，喜歡這或那個偶像明星，而這一定會讓明星收到的“粉絲稅”更加膨脹。這個稅，其實並不是運營方騙你，而是粉絲本身確實通過這個明星 IP，獲得了快樂。但物質享受帶來的快樂邊際效用遞減的時候，你會將更多的金錢貢獻給偶像。

第六、IP 或許是虛擬的，刻意以打造的，但快樂是真實的，這或許是人格化偶像模式得以瘋狂賺錢的倫理根基。你不能說吃一頓大餐、穿一件新衣服、買一輛行車、住進一間新房子，得到的快樂是快樂，而喜歡看一個藝人，得到的快樂就不是快樂。這兩者沒有高低貴賤之分。事實上，物質享受帶來的價值是價值，精神享受帶來的價值也同樣是價值。如果你閉上眼睛仔細感悟這個世界的本質，會發現都是一樣一樣的。要說智商稅，你到哪裏都是被收智商稅，只是包裹了不同的外衣。吃飯被收稅，買車被收，買房被收，看電影電視也一樣被收，對嗎？然後就精神產品這個小領域來說，也是一樣的。你說娛樂明星在收智商稅，沒有創作什麼實際價值，那麼請問體育明星創造什麼實際價值了？體育的本質和影視是完全一樣的，完全就是表演，你真覺得足球踢得好，籃球打得好，對改變人類生活有什麼作用？還不是一樣套路，一樣的粉絲運營手段？正因爲這樣，我作爲一個 80 後男性，纔敢說，喬丹並不比鹿晗更偉大，因爲兩者本質是一樣的，不能因爲我兒時喜歡喬丹，現在就鄙視 95 後小女生喜歡鹿晗。這兩者有什麼本質區別？你告訴我。你說喬丹代表男人，鹿晗代表娘炮，這只是你一廂情願的想法。因爲有大衆的需求，纔有了鹿晗，人格化明星是受衆羣是相輔相成的。喬丹和鹿晗這個兩個 IP 的本質，都是給一部分受衆的青年時代帶來和希望和快樂，就如同佛祖和耶穌的 IP 都起到了撫慰受衆心靈、引導覺悟的作用。如果鹿晗的受衆比喬丹更多，那麼鹿晗賺錢比喬丹更多是應該的，和其有沒有肌肉，會不會運動，沒有關係。

大概就先說到這裏。明星藝人，以及構建在明星藝人之上的影視公司，嚴格意義上都不是一個適合風險投資去合作的標的，但名藝人瘋狂圈錢背後的邏輯是可以給我們啓發的。我一直堅定地認爲人類虛擬化之路不可阻擋（虛擬現實能解決一切問題，包括最終解決明星本人和 IP 合一的問題），精神消費的佔比會不斷提高。
