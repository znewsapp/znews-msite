「留你工作？我不如多花點錢重新招一個人進來」

「寧願花 11K 重新招人，也不願意花 9K 留住老員工」是真的嗎？爲什麼會出現這一現象？

題幹裏面其實已經道破了答案。關鍵很簡單：

同樣一個崗位上，不同員工的工作能力是不一樣的。

畢竟，公司招人是帶有很強不確定性的。大家應聘時，每個人都會試圖把自己的能力說得很強，所以公司光靠簡歷和麪試很難確定求職者的工作能力。這也是爲什麼很多公司有試用期的原因。但是，試用期畢竟也太短，很難暴露問題。

舉個簡單的例子吧。假設市場上有三種類型的求職者，能力也有所不同，各佔三分之一。A 類員工的工作能力是 100k，很強；B 類員工的工作能力是 60k，一般；C 類員工的工作能力是 0，很差。假設公司年薪是 50k， 那麼 A 類員工每年可以爲公司貢獻 50k 的利潤；B 類可以爲公司貢獻 10k 的利潤； C 類每年爲公司貢獻 50k 的虧損。這個設定，應該比較簡單易懂了。

那麼假如公司能夠分清這三種類別，肯定會盡量多招工作能力強的 A 類人才，最不濟也招一些 B 類人才。問題是，每個人的能力，公司是看不到的，而 B 類 C 類會假裝自己是 A 類，從而得到公司的青睞。要等到公司真的僱傭了他們，才發現有些人是無法勝任工作的。這個問題怎麼解決呢？

當然，公司可以通過學歷等等信號，來對員工水平進行篩選；也可以通過績效考覈的方式，對員工能力進行覈查。可是如果沒有這些手段，或者這些手段只是表面工夫，沒有起到應有的作用的話，那麼裁員和重新招聘就很重要了。

我們接着上面的例子。公司在上一年招聘之後，發現自己招了很多 C 類員工，老闆很生氣。又到了畢業季，老闆發現這一屆畢業生的素質有所提高，A、B、C 類員工的比例從各三分之一變成了 40%，40%，20%。那麼公司最多可以出多少年薪，來從新的一屆畢業生當中招聘呢？

同樣的道理，改個數字，就是這道題的答案了。

公司不願花 9k 留住的老員工，很可能是工作能力不足的 C 類；而花 11k 重新招的新員工，很可能是工作能力爆棚的 A 類。

每一屆求職者的情況不同，所以也不能去指責這種做法是缺乏人性的。畢竟，公司需要的是你的工作能力，而不僅僅是你的輩分或資歷。雖然工作能力很可能會隨着經驗增長，但是如果增長得不夠快，或者你無法證明你的能力有增長的可能，那麼你很可能就會在職場中被淘汰。

最後，我要表明一下立場。我們這些窮博士其實也是這種現象的受害者。

聽說我們後面幾屆入學的學弟學妹，每年給的獎學金都比我們這屆多很多。主要是因爲從某一屆開始，哈佛和斯坦福給的錢漲了，所以芝加哥爲了搶生源不得不水漲船高。可是，我們這些之前入學的人，卻還只能拿原來那麼多的獎學金...

當然，這個原理和我這裏說的還不太一樣，就不展開說了。多說都是淚！
