年度熱門 · 我們家貓咪死了，罪魁禍首竟然是我點的蚊香

點蚊香對貓會有傷害嗎？

貓何止是不愛聞這個味兒，根本是這個味道可能會害死貓的……

蚊香有毒的事情大家多多少少都略有所聞，但是作爲已經取得生產許可證號、獲得產品標準號面向市場銷售的商品，理論上來講，只要根據產品說明書控制使用劑量，人身安全應該還是有保障的。

問題就在於，許多鏟屎官忽略了在這些安全評價體系中，所謂的“安全性”都是以人作爲對象進行評估的。而對於喵星人和汪星人，由於它們的體重與生理代謝特點與人並不完全相同，即使是降低劑量、或使用含有天然成分的驅蚊產品也可能會對它們造成致命傷害。

驅蚊酯（菊酯），便是這樣的一種“貓毒藥”。

請儘量避免或警惕使用以下驅蚊 / 蟲物品

目前，市售的各類驅蚊產品不外乎都含有這五種主要成份：ef="">DEET (避蚊胺)、Picaridin(派卡瑞丁，羥乙基哌啶羧酸異丁酯)、Oilof LemonEucalyptus(檸檬桉葉油)/PMD(檸檬桉葉油提取物)、IR3535(伊默寧)及菊酯類化合物(如除蟲菊酯、百滅寧、滅白可等)。

在這之中，菊酯類化合物毒性最大（毒性越大越有效），而價格最爲低廉，是目前市場上幾乎所有非接觸類蚊香盤、電蚊香的主要成份。

菊酯化合物種類繁多，如除蟲菊酯、氯菊酯（即百滅寧）、胺菊酯、氯氰菊酯（即滅白可）、溴氰菊酯、右旋反式烯丙菊酯等，這些衍生物都屬於世界衛生組織推薦的可用於製作蚊香、防治衛生害蟲及其媒介的農藥類物質。因此只要按說明使用，對人來說就不會有中毒的危險。

之所以說對人類相對安全，是因爲人的肝臟具有可以將這類化合物代謝爲無毒或低毒物質排出的能力。與人類體質相似，汪星人雖然體型較小，但肝臟也具備清除這種菊酯類物質的能力，因此有許多汪星人的滅蚤項圈也含有較低濃度的菊酯類物質。

然而，

喵星人沒有代謝菊酯的能力！

也就是說，對於喵星人來說，對菊酯類蚊香、電蚊香的耐受性遠遠遠遠遠遠遠遠低於人類和汪星人。我們也可以理解爲，如果人和汪星人體吸入少量的菊酯，那麼是能夠緩慢清除掉的，不對身體造成傷害；而喵星人即使吸入少量菊酯，也幾乎完全停留在體內，很難被代謝掉，從而引起蛋白質變性、腦部神經系統受損、器官衰竭甚至死亡（其實就是喝農藥後的症狀）。不僅對喵星人造成長期傷害，喵星人中毒後也很難恢復。

菊酯類化合物的基本構架類似、結構各異。

在這些化合物中，二氯苯醚菊酯和氯氰菊酯幾乎完全不能被喵星人代謝，對喵星人來說，是堪比砒霜一樣的存在。

二氯苯醚菊酯（Permethrin）又稱苄氯菊酯 / 撲滅司林 / 百滅寧 / 氯菊酯，氯氰菊酯（Cypermethrin）又稱又稱滅百可、安綠寶等，二者都是盤式蚊香、電蚊香中常用的驅蚊物質之一。

此外，一些市售犬用除蚤圈或驅蟲滴劑也是用菊酯製成的，如廣受歡迎的德國拜耳拜寵爽犬用體外驅蟲滴劑和美國 K9 Advantix® II 犬用體外驅蟲滴劑都含有二氯苯醚菊酯。可想而知，這些對人和汪星人相對安全的產品如果誤給喵星人使用，和給喵星人吃狗糧的嚴重程度可是完全不一樣，後果將不堪設想。

K9 Advantix®系列犬用驅蟲滴劑主要有效成分爲吡蟲啉、二氯苯醚菊酯和吡丙醚，商品在外包裝上註明有“避免對貓使用”字樣。

與過量吸入蚊香而導致人感到不適的症狀相似，“中毒”後的貓咪按程度的輕重可以表現爲：

1、流口水

2、走路歪斜，失去平衡能力（其實是頭暈吧）

3、興奮過度，後期出現抽搐

4、嘔吐

5、食慾不振

而減小蚊香用量以及加強通風的方法除了使貓咪由急性中毒轉爲慢性中毒以外，可以說沒有任何用處。而高頻次使用菊酯類蚊香，或鏟屎官不幸買到了有效成分未明的，和實際濃度高於安全值的“三無”劣質蚊香，或家裏的通風狀況不好，或貓咪幼小老弱、體質不好，都可能出現開頭所述貓主子被毒死的悲劇。

別看了，不是質量問題。普通的市售電蚊香產品對人類都是安全的，問題出在喵星人體質特殊。可是那位宣傳除蟲菊酯對貓狗都適用，並且安全無毒的賣家，你的良心不會痛嘛！

值得注意的是，其實貓咪代謝許多有毒物質的能力都十分有限，雖然愛貓家庭可以用含有避蚊胺、檸檬桉葉油類的物質塗抹在衣物內代替菊酯，但倘若貓咪不慎將這些物質舔入或者吸入，則同樣會導致中毒的症狀。

貓咪如果出現了以上症狀，鏟屎官應立刻將貓咪送到獸醫處進行緊急醫治，否則將會導致貓咪身體不可逆的傷害、腎衰竭甚至死亡！

但是，由於喵星人體質特殊，即使是最專業的治療也只是在短時間緩解貓咪的中毒症狀，長期清理毒素需要多次透析，對貓和鏟屎官造成的身心傷害都是不可估量的。

因此鏟屎官們應該從源頭抓起：

1. 養貓的家庭應格外注意或拒絕使用含有菊酯類物質的各種物品，儘量通過使用電蚊拍、蚊帳、紗窗等物理防蚊；

2. 在主人塗抹 DEET 或其他驅蚊產品後注意洗手，注意和貓主子保持距離，儘量不要讓貓咪把藥蹭到毛上去；

3. 不要隨便把犬用驅蟲藥、除蚤圈往貓主子身上套;

4. 更不要隨意從網上購買“三無”的不表明成分、用量、尤其是價格低廉的貓星人除蚤圈，甚至對於大廠生產的正規貓用驅蟲滴劑在使用後隔天也最好通過洗澡去掉，確保貓主子的健康！
