懷孕期間，有哪些常見的錯誤常識？

我們有哪些錯誤的常識？

開門見山地說，孕期是可以用藥的！！！

月初，一個朋友帶着他懷孕 6 個月的愛人找到了我，說是已經感冒咳嗽半個多月了還沒好，現在緊急求助來了……

我幫着看了一下，咳嗽，咽痛，鼻塞流涕，又讓她查了一個血常規，白細胞，中性粒細胞比例都有些偏高，聽了一下肺部，還好沒什麼異常，嗯，確定是個上感了。

於是，我又帶着他們去了產科，產科的同事看了一下，建議使用抗生素，聽到要用抗生素，我朋友頭搖的跟個撥浪鼓似的……他愛人最好不要用藥，尤其是不要使用抗生素……

我和同事一臉懵逼，告訴他們有些藥物，比如頭孢，青黴素類藥物在孕期還是可以使用的，尤其是長時間感染或者感染較重的時候。

即便如此，他們夫妻兩還是不太能接受，我同事又跟他們說：你們現在必須使用抗生素，第一，他們家已經過了前面三個月了，現在屬於相對穩定期，用藥的風險比較低；第二，頭孢屬於孕期 B 類用藥，是相對比較安全的，可以使用。

經過反覆勸說，朋友夫妻才決定使用，於是我們給他們配了一盒頭孢地尼。

放假前，又和他們聯繫了一下，他愛人已經康復了。

下面，我想再普及一下關於孕期用藥的知識。

在孕期用藥的分級中，FDA 根據動物實驗和臨牀用藥經驗對胎兒致畸相關的影響，將藥物分爲 A、B、C、D、X 五類。

1、A 級

A 類藥物，是指在臨牀研究中，未見到對胎兒產生危害的跡象，該類藥物對胎兒的影響甚微。

屬於 A 類藥的藥物極少，維生素屬於此類藥物，如各種維生素 A、B、C 等。

不過，值得一提的是，正常範圍量的維生素 A 是 A 類藥物，而大劑量的維生素 A，每日劑量 2 萬 I U，即可致畸。

2、B 級

B 類藥物在動物研究中（未進行妊娠期婦女的研究），未見到藥物對胎兒的不良影響；或在動物研究中發現藥物有副作用，但這些副作用並未在妊娠婦女身上體現。

屬於 B 類藥的藥物亦不很多，可喜的是日常用的抗生素均屬此類。如所有的青黴素族及絕大多數的頭孢菌素類藥物都是 B 類藥物，常用的氨苄青黴素、頭孢拉定、頭孢三嗪和重症感染時搶救用的頭孢他定等都是 B 類藥。另外，雙氯芬酸、布洛芬、甲硝唑、潔黴素、氯林可黴素、紅黴素、呋喃妥因也是 B 類藥。

3、C 級

C 類藥，是指動物研究證明藥物對胎兒有危害性（致畸），但尚無妊娠婦女的研究；或尚未對妊娠婦女及動物進行任何研究。

本類藥物只有在權衡對孕婦的益處大於對胎兒的危害之後，方可使用。

屬於 C 類藥的藥物是較多的。某些抗生素，比如喹諾酮類藥物；大部分的抗病毒藥，如阿昔洛韋；部分抗癲癇藥和鎮靜劑，如乙琥胺、非氨脂、巴比妥、戊巴比妥等；某些利尿劑，比如呋噻米、甘露醇均爲 C 類藥

對 C 類藥物的使用要謹慎，如果有可以替代的藥物則選用替代的藥，否則在權衡利弊後，向患者或患者家屬說明選用該藥的理由。

4、D 級

D 類藥是指，有明確證據顯示，藥物對人類胎兒有危害性，但儘管如此，孕婦用藥後絕對有益（例如用該藥物來挽救孕婦的生命，或治療用其他較安全的藥物無效的嚴重疾病）。

由於已有實驗和臨牀上的證據，D 類藥在妊娠期，特別是在早期妊娠階段儘可能不用。

D 類藥中，比較常見的就是抗生素中四環素族和氨基糖甙類藥物。這些藥物在妊娠期儘可能不用，例如屬於氨基糖甙類藥物的鏈黴素，可能損傷第八對顱神經而發生聽力問題。

5、X 級

對動物和人類的藥物研究或人類用藥的經驗表明，藥物對胎兒有危害，而且孕婦應用這類藥物無益，因此禁用於妊娠或可能懷孕的患者。

大名鼎鼎反應停（婦女在孕早期服用此藥以減輕妊娠反應）就屬於 X 類藥物，很多服用反應停的婦女所產下的胎兒出生時有上肢短小，下肢合併而呈海豹狀故稱之爲海豹樣畸形。

這裏又要提到維生素 A 了，當維生素 A 大劑量口服時，由於其致畸作用，此時的大劑量維 A 也屬於 X 類藥物。
