大誤 · 我已經看到了財務自由在向我招手

我本來是不想寫這個的，昨天看了曹政的文章《不要試圖挑戰人性》，感觸挺深的，曹大說他 2018 絕不碰 ICO，我恰恰相反，不但要碰我還要帶你們一起碰。ICO 現在簡直是太火了，我平均每星期收到 1-2 篇 whitepaper，絕大多數，可以說是 100% 都是忽悠人的，我不當韭菜也會有別人被收割，莫不如我今天就給你們指條路——幹嘛參與別人的 ICO 啊？你可以自己搞啊！

廢話不說，首先去以太坊下載一個錢包。Ethereum Project

恭喜你，你已經成功克服了你 ICO 道路上最大的技術難關，勝利在望，會所嫩模在向你招手！

這個時候系統會讓你輸入三個參數：

Token synbol：就是幣的符號，比如比特幣是 BTC，以太坊是 ETH，咱們嫩模幣當然是 NMB 了！

然後下面有個藍色的 deploy,點了這個 deploy,嫩模幣就正式發佈了————這裏有個條件，就是錢包裏要有少量的 ETH，作爲執行合約的 Gas，大概是 0.00 幾個 ETH 就夠了，也就幾美元到幾十美元的事兒。

好了。

完成了。

如果你完成了如上所說的步驟的話，那麼你成功的在這個世界上，基於以太坊網絡，創造了一種新的加密貨幣————如果這破玩意兒能稱爲加密貨幣的話……

我大概解釋一下啊這玩意兒是啥：以太坊網絡和只能合約，支持一個 use case 就是用戶可以通過以太坊來發行自己的"Token", Token 是什麼呢？你可以理解爲現實生活當中的“積分”，對，比如加油站洗車店會員卡積分，樓下發廊 Tony 老師讓你辦的衝 2000 送 1000 的美髮會員卡，奶茶點送你的蓋滿 10 個張送一杯的集戳卡，幼兒園老師給小朋友的小紅花……

這一切的一切，都是可以在以太坊用很簡單的只能合約代碼搞成一個“token”,然後 Token 可以通過以太坊網絡轉賬， 轉賬的時候消耗少量的以太坊 ETH 做 Gas。Token 也不需要錢包——使用以太坊錢包就好，錢包地址也是以太坊的地址，錢包祕鑰也是以太坊的祕鑰，區塊鏈用的就是以太坊的區塊鏈……

說了這麼多，就是想讓你明白，Token 這破玩意兒如此簡單，發行如此容易，沒有成本，完全是基於以太坊網絡，沒有任何自己的底層技術，基本上 20 分鐘就能搞出來一個，供應量隨便填。

我要說的是：現在有好多所謂的 ICO，就是把這 Token 拿來當幣賣……

如上所說創造出來的 Token，跟比特幣，以太坊，瑞波這些真的“加密貨幣”完全不是一個東西——我不是要給比特幣以太坊這些數字貨幣站臺——我一開始以爲，要搞 ICO，起碼要像比特幣那種，自己有區塊鏈，有底層技術，分佈賬本，錢包，nodes 都自己實現吧——雖然這也不是什麼難事兒，畢竟都是開源的，但起碼這還是有一定成本和壁壘的，結果，這幫騙錢騙瘋了的，連這都不搞，直接像我上面發佈嫩模幣那樣，利用小白韭菜們的無知，把 Token 拿來當成幣賣……

打開 All Tokens | CoinMarketCap 看看哪些你以爲和比特幣，以太坊一樣的加密貨幣，其實是和嫩模幣 NMB 一樣的 Token 呢？

有沒有很眼熟啊？有沒有很心慌啊？你看 Platform，就是平臺，多數都是以太坊平臺，估值……50 多億美金？80 多億美金？

你們覺得嫩模幣 NMB 估值應該多少啊？這麼搞下去會不會幣太多了，嫩模都不夠用了啊……

好，現在幣有了，接下來就是 ICO 難度最大的部分了：

寫白皮書啊！

有沒有發現現在白皮書天馬星空，包羅萬象，無所不能？爲啥？本質上這玩意兒就是“積分”，能用的上”積分“的地方都能給套進去啊！機械製造，生物製藥，航天科技，基因工程，物流運輸，文化創作，陰陽五行，一帶一路……反正你想怎麼寫怎麼寫，這部分就非常考驗 ICO 團隊的吹牛逼功力，白皮書寫的不好，給觀衆的想象空間不夠，基本可以判斷，這個團隊平時缺乏詐騙經驗……

有了幣，有了白皮書，就可以拿去忽悠人了，基本上就是說 ICO 前期，讓對方給你打以太坊 ETH，然後你給他發你的嫩模幣 NMB……他給你以太坊，你給他嫩模幣，他給你 ETH，你給他 NMB……他的以太坊得是 1000 美金一個真金白銀買的，你的 NMB 可是上面隨手填出來的。過不過癮？爽不爽？

這玩意兒，薄利多銷，騙到 50 算 50，騙到 100 算 100，團隊別露臉，白皮書上不寫名，寫名也別寫自己名，找兩個外國人做 adviser，反正韭菜們也不會去驗證。核心團隊叫基金會，比如咱們負責操作嫩模幣的基金會，可以起名叫 NMB FOUNDATION。收到 ETH 直接變現分錢走人，或者另起爐竈，NMB ICO 成功完成了，再搞下一個唄，市場需要細分，嫩模有的醜有的俏，針對醜嫩模搞一個醜嫩模幣 CNMB，俏嫩模的叫 QNMB，或者叫嫩模 2.0 幣 NM2B，聽上去就有互聯網時代感……

區塊鏈確實是個好技術，有很多潛力和空間，但究竟有沒有好到顛覆世界，有沒有好到隨便發個 tokn 就能值十億八億的美金，有沒有好到能讓人打着“區塊鏈”，“智能合約”，“加密貨幣”的幌子搞詐騙和非法集資能逃脫法律責任，有沒有好到能讓你買一個 Token 就賺個幾百幾千倍。大家自己判斷。
