現實中存在「福爾摩斯式」推理嗎？

朋友，你聽說過女人的第六感嗎？

我問誰啊。

老公說單位組織活動剛認識的幾個同事，邀請我們一起去他們家吃火鍋。

然後我就不淡定了，剛認識就去過妹子家了？

發小一臉懵逼地看着我：你從哪看出來他去過別人家啊？

發小當即表示有理有據，令人信服。

老公證實了自己活動期間確實幫人家拉過古箏，送貨上門。

 

還有一次老公來接我，副駕駛靠背往前挪了一點。我 1 米 68，生活在以大漢著稱的山東省，不敢相信身邊哪個男的會比我矮。

半年前，我們這座只有着不到 4 萬實有人口的小縣城發了一起巨案：某企業會計遭遇了電信詐騙，白白拱手相送了 890 萬元人民幣。

可喜可賀的是，案子在上級、上上級和上上上級的督辦下很快就破了（金額越大越容易破）。我們把嫌疑人都抓回來了，就放在我們辦案區。

人手不足，臨縣刑警過來幫忙取證，把臺灣人帶進帶出的好不熱鬧。有幾個臨縣刑警因不熟悉我縣辦案區密碼，準備打電話問人呢，誰知嫌疑人張口就說：試試 1900。

臨縣刑警一試，鐵門應聲而開，驚得臨縣刑警高呼：三小？？？！！！

問臺灣人怎麼知道辦案區的密碼，臺灣人娓娓道來：

一，當日民警帶我進入的時候，我聽到她共按下了四顆按鍵，每顆各有一聲回饋音“嘀”，即密碼有四位數。0 至 9 十個數字按四位（可重複）排列（* 鍵和#鍵爲編程鍵，暫不計算在內），一共有 10×10×10×10=10000 種組合，按理來說等我一個一個試試出密碼的時候，臺灣也該光復大陸了。

二，我能聽出四顆按鍵的回饋音的節奏爲“嘀——嘀—嘀嘀”，即“A——B—CD”。根據節奏得知，按 A 鍵與按 B 鍵之間時間間隔（位置）最長，按 B 鍵與按 C 鍵之間時間間隔（位置）很短，按 C 鍵與按 D 鍵之間時間間隔（位置）極短，甚至可猜測爲一個鍵。

三，路過鍵盤時通過快速觀察得知，0 至 9 共 10 顆按鍵中磨損最嚴重的按鍵分別爲 0、1、9，只有三顆按鍵，與 CD 鍵爲一顆按鍵的猜想不謀而合。當 CD 爲一顆按鍵時，“A——B—CD”公式可進化爲“A——B—CC”，且 C 只能是 0 或 9，因爲這樣才能與以上第二條相符，因爲 1 無論是與 9 或是 0 都非常遠。進一步推得 A 鍵應當爲 1。

四、分組討論：當 C 鍵爲 9 時，密碼爲 1099、0199,帶入第二條、第三條最終爲 1099；當 C 鍵爲 0 時，密碼爲 1900、9100，帶入第二條、第三條最終爲 1900。

五，根據人類一般正常順序邏輯，應當首先猜測 1900，因從感官上看，數字較爲整齊。不料成功解鎖。
