別隻會悶頭聽歌了，和女朋友吵了架，這幾點是你還能做的

戀人在爭吵的冷靜期間，適合做些什麼？

世界上沒有比愛更艱難的事情了。

---- 《霍亂時期的愛情》

簡單的說，在爭吵的冷靜期間，我想有三件事情是可以做的。

第一就是自我照顧。

嘗試照料自己的情緒。把自己快樂與否的權力完全的放在別人身上是一件非常危險的事情。嘗試理解到底我在什麼地方受到了傷害，到底 TA 怎麼傷害到了我，這一點非常重要。

很多時候人們只記得疼的狀態，而忘了疼的內容。我是感到被忽視、或者被拒絕了嗎？我是感到被踐踏自尊了嗎？我是感到被要求了嗎？

嘗試問問自己，有什麼是我能自己爲自己做的，讓自己受到傷害的部分，情緒上能感覺好一些？也許是一個朋友的陪伴，也許是投入一個愛好的片刻放鬆，或者是一份美食？

你是自己面對一切困苦艱難的基礎，先彆着急考慮別人。首先看你，再看其他。

第二是思考。

吵架總有原因，事情發展總是有個進程的。思考一下你們是如何從相親相愛的比翼鳥，變成了相恨相殺的伏地魔和哈利波特。一般來說，大部分的吵架都是基於事實而生，但爭執超越了事實的關係角色問題。用句調侃的話來說，就是藝術源於生活高於生活。

吵架更深層次的原因多少都會涉及彼此對於愛情的期待，對於對方在愛情中扮演角色的期待，以及對方對待自己的方式等一系列問題。而這多半都是經過一段時間累積，而非一次性出現的。

舉個簡單的例子，女孩抱怨男友總是和自己調侃其前女友的身材並且大加評價。這看上去是兩個人因爲一段不合時宜的對話而產生的問題。但其核心在於，女孩不滿男孩總是把自己當哥們一樣相處，而不是當做伴侶那樣尊重，認爲他不能扮演自己所期待的有擔當的成熟男性角色。

因此嘗試思考衝突問題的核心，理解問題的產生是非常必要的。這能讓你們真正走進關係問題中去，而不是簡單地就現實的雞毛蒜皮爭吵。

第三是積極準備重建溝通的內容和方式。

很多戀人爭吵冷靜期後就直接是甜蜜的擁抱、像什麼都沒發生一樣的繼續相處了。這既不解決問題，對於關係的促進和積極發展也沒有什麼建設性的幫助。就像是一個爆胎的自行車，你可以嘗試推着走，你可以嘗試湊合騎，你甚至可以嘗試反覆打氣。但是車胎破損的地方，不去修補，終究是漏了的。

千萬不要把標準僅僅侷限於“嗯，最近 TA 還不錯，沒有繼續鬧騰”的 0 分標準上。從負數到 0 不是勝利，到達你期待的愛情分數纔是重點。嘗試在冷靜期準備關於你們問題核心的談話，對於把衝突轉化爲關係進步的力量是非常重要的。感情問題的改變離不開反覆的溝通和交流。你是怎麼想的，我是怎麼想的，這之後我們該怎麼辦。這不代表一次溝通就能讓你們找到完美的答案。溝通既是交流的過程也是共同面對的過程。很多時候共同面對、共同體會困難本身就是一種解決和親密。

以上三點希望能對題主有所幫助。愛是一件艱難的事情，也因爲難，所以千甜萬甜。

所有的戀人都會吵架，即使是那些最爲成功的戀人也不例外。研究發現，伴侶們發生矛盾的頻率也很頻繁——戀愛期間的伴侶們，平均每週會發生 2－3 次衝突。在一項爲期三年的研究中，夫妻報告他們每個月都會經歷 1-2 次“不愉快的爭論”，並且這個頻率在三年中並沒有發生變化（Millder & Perlman, 2010）。

與其他類型的關係不同，在很多情況下，戀人之間的爭吵並不是因爲要爭出誰對誰錯，而是一種表達感情的渠道。在某些時候，爭吵甚至還會給戀愛關係帶來一些好處——它讓雙方產生一種重新的聯繫，也能夠幫助戀愛雙方減輕心理壓力(Godek, 1997)。可以說，情侶間的爭吵也可能是健康的，有利於關係發展的。

那麼，在這樣爭吵之後的冷靜期，我們可以做些什麼，以利於關係的“修復”和進一步發展呢？

1. 暫時不要急着去和彼此接觸，你們需要平復自己的心情

在爭吵之後，人們往往會體會到憤怒、受傷、困惑等情緒，這些都是很正常的。然而，如果你帶着這些情緒就急於和你的伴侶接觸的話，你很可能會無法控制自己的情緒，說一些自己本來不想說的話，進而傷害到對方。爲此，你可以試着獨處，比如一個人的短途旅行。但是，這並不意味着你要和伴侶冷戰(Feintuch, 2017)。

2. 思考你們究竟想通過這次爭吵達到什麼樣的目的

在冷靜下來的時候，你們各自都可以去思考，自己究竟爲什麼要和伴侶爭吵，你們希望彼此關係有一種什麼樣的走向(Feintuch, 2017)。比如，你希望伴侶能夠更多的關注你的感受。當你想明白這一點之後，其實你會發現，“吵（贏）架”並不是自己真正的需求，而可能只是一種方式或者只是在溝通過程中發生的“狀況”。之後，你便可以嘗試以別的“方式”來表達你真正的需求。

3. 多從對方的角度來思考問題

你可以嘗試從對方的角度考慮問題，這可以幫助你看到你們真正的分歧或者引起你們爭吵的“癥結”所在。如果你覺得此時很難換位思考（可能你還是感到很生氣），你可以想象自己是一個關心你們兩人的人（比如，你們最好的共同朋友），試着從一個第三方的視角去思考，你伴侶言語背後的看法、觀點、需求是什麼(Weiss, 2016)。

4. 照顧好你自己

有些人會因爲和伴侶吵架而不停的傷害和懲罰自己。他們可能是因爲悔恨自己說了一些不該說的話，或者可能會希望用自己的痛苦來博取對方的同情。不過這些做法往往都對你自身的身心健康，以及爭吵後關係的修復沒有助益 (Feintuch, 2017)。

事實上，你需要做的是，在吵架的冷靜期仍然照顧好你自己，比如按時吃飯和睡覺，多和其他的朋友交流等等。

說到這裏，我們還想強調的一點是，當有些爭吵已經是“不健康”的，不利於關係的穩定與發展的時候，例如，對方總是在爭吵中口出惡言、爲了“吵贏”不惜傷害你，此時，你在冷靜的時候，你就需要認真地思考如何離開這段關係了。

那麼，你需要分辨什麼樣的爭吵是不健康的，是讓你需要開始考慮離開的(Florentine, 2016)。

1.你們之所以吵架是因爲關係當中的一方經常性的不誠實、背叛。

2. 你們反覆多次爲了同一件事情不斷的爭吵。這從某種程度上說明了，你們雙方都缺乏共同解決它的動機或方法（不要總試圖指責對方不改變，你是否想過自己爲什麼總試圖強迫對方改變呢？）

3. 你們對未來的規劃、對人生的信念 / 態度南轅北轍。

4. 在爭吵中，你的伴侶會對你採取精神 / 肉體上的虐待，比如選擇要挾、貶損、冷戰、暴力等方法。
