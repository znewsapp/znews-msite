大誤 · 皇上，這可是寶貝啊

如果穿越到古代，除了衣服，只能帶價值 100 元東西的話，帶什麼東西最好？

準備工作(๑•̀ㅂ•́) ✧

1.買幾包水晶寶寶（或者說泡大豆），就是你上小學的時候玩過的那種。

一包 500 粒，只要 0.1 元。

2.還有魚缸造景用的熒光砂

淘寶一個一毛。

3.玻璃彈珠，三個一元

買三個就夠了，用途後面會講。

4.五瓶風油精

5.隨便打印一份地圖，芬蘭瑞典挪威冰島丹麥隨你選，魔獸地圖也行。

6.打印出一份詳細的大事年表，除歷史事件外還要記錄地震等災害的時間地點，並打印出歷朝歷代所有皇帝的出生日期。最好縮印。

7.阿司匹林，你值得擁有

一盒 14，多備幾盒。

8.手紙

9.純淨水

10.一次性打火機

11.衣服穿厚些，短袖上套長袖，長袖上套羽絨服，羽絨服上套羊毛大衣。

一定要穿秋褲！

12.大紙箱

13.玻璃碗

14.刻個橡皮章

穿過去之後(๑•̀ㅂ•́) ✧

1.如果天氣熱，把羽絨服長袖大衣秋褲之類的脫掉並裝箱。（冬季跳過）

2.看看自己到了哪個朝代，把與此朝無關的史料燒掉。

3.用水將大部分泡大豆 / 水晶寶寶 / 海洋寶寶（即圖一）泡發，裝至玻璃碗中。

記得給自己留幾顆。

4.聲稱自己是 xx 國使者，要求進宮面聖，有印章爲證。

國家名字可信口胡編，切記不能是太近的國家。

5.面聖後，獻上之前打印的 xx 國地圖，並將玻璃碗中的泡大豆獻給皇上。

花言巧語，可自行發揮。

例：長安以東三萬裏曰子虛之山，其狀如扇，臨於荒海之上。山中有湖，湖中有奇蚌，其名曰烏有蚌。每逢盛世，奇蚌吐珠，珠華四照，其名曰軟玉。

除此之外，也可以把泡大豆說成鮫人之淚 / 神魚之卵等。

6.獻風油精

例：又西北四百二十里，曰 x 山，其上多青木，其味如飴，食之不飢。剖樹取汁，得瓊漿。其色碧綠，有異香，可驅蚊蟲，君子服之，以御爲祥。

7.帶上皇帝賞賜的銀兩，在京城裏住下。在未來的皇帝降生的前一天再次進宮，告訴皇帝自己的泡大豆（軟玉 / 鮫人之淚 / 神魚之卵）在今天變成了碧玉，這是天降祥瑞。

並將熒光砂和玻璃球獻給皇上。

次日皇子（即未來的皇帝）出生，現任皇帝也許會認爲你有神力，能當個御用風水師啥的。（哪有這種官啊！摔！）

8.在皇帝 / 后妃 / 皇子患病時入宮，聲稱自己能用法術治好病。

燒張符，加入放了阿司匹林的水中，讓病人服用。

病癒後，皇帝對你並將更加信服。

（這一條風險很大，萬一病沒好......）

9.在地震到來的前幾天，聲稱自己夜觀天象，某某某處必有大災。

之後可一直走神棍路線，保你一生榮華富貴。
