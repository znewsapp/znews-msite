越來越多大導演開始放棄 3D，2D 又要回歸了嗎？

2D 電影重新迴歸，3D 電影何去何從？

3D 能不能像聲音和色彩一樣，成爲電影敘事的必要手段呢？

自從卡梅隆用《阿凡達》“重新發明”3D 電影以來，3D 幾乎成爲了大片的標配。伴隨着票價和影院設備的升級，現在要拍攝、上映、觀看一部非 3D 的大片，似乎已經成爲一種“政治錯誤”的舉動。在這種背景下，諾蘭的《敦刻爾克》依舊堅持用 2D 上映，不僅僅需要勇氣，還需要一種藝術的自信。

在這背後的，是 IMAX2D 的《敦刻爾克》的不俗票房。當然，不能否認的是，諾蘭很有可能利用了自己的影響力，力保影片用 2D 格式上映。但是越來越多的導演，也正在漸漸拋棄 3D——尤其是 3D 並不能帶來更好的觀影感受的時候，他們對於 3D 的放棄，似乎顯得正確而且大公無私。十月份即將登陸大銀幕的《銀翼殺手 2049》，雖然是一部高科技、高概念的大片，但卻依舊選擇了 IMAX2D 格式。

《阿凡達》

有一組數據可以表明，爲什麼導演正在漸漸放棄 3D。2016 年，好萊塢一共製作了 68 部 3D 電影，數量創造了歷史新高。但是去觀看 3D 電影的觀衆人次，卻同比減少了 8%。兩廂綜合，3D 電影市場的縮減，是一個不爭的事實。

有很多原因造成了這個局面。不僅僅是過暗的銀幕亮度，笨重的 3D 眼鏡以及超過 2D 影片 30%的票價。最爲關鍵的是，3D 在很大的程度上，並沒有提升觀影質量。相反，在很多場景中，3D 反而成爲了累贅。拖影、模糊、抖動，種種觀影過程中的 bug，已經嚴重影響了觀影體驗。在一定程度上，放棄 3D，反而是更好的服務於觀衆。

其實早在 2012 年，英國媒體就提出了“3D 熱潮銳減”的觀點。只不過彼時 3D 正如火如荼，誰也沒有注意到這一點。當時間轉到 2017 年，3D 電影已經經過了斯科塞斯（《雨果》）和文德斯（《皮娜》）這種級別導演的實踐之後，似乎已經成爲了商業上的“刻工求意”，而不是電影語彙的一種拓展。

3D 究竟有沒有參與敘事，並且成爲表情達意的電影手段呢？答案在目前這個階段，基本是否定的。因爲無論是在《變形金剛》、《雨果》還是在《地心引力》抑或是在《少年派》之中，都沒有成爲電影的語彙。所以，IMAX 公司決定削減 3D 電影的數量和比重，來重振公司的口碑和業績，也是有理可尋的。

在未來，3D 是不是會成爲電影語彙的一種，並像聲音和色彩一樣，成爲一部電影“所必須”的部分，這依舊是一個未知數。

在電影語彙得到拓展的時間節點上，很多諾蘭無法比肩的導演和攝影師，都曾經“倒行逆施”地開了歷史的倒車。有聲電影出現的早期，偉大的卓別林和更偉大的愛森斯坦，都曾經極力地反對“聲音”這個電影元素。但當時間沉澱，錄音質量不再是技術掣肘的時候，這兩位大導演並沒有固執己見地拋棄有聲電影，反而留下了影史上著名的《大獨裁者》和《伊凡雷帝》。

《大獨裁者》

而色彩，也經歷了類似的過程。費里尼和羅西里尼的御用攝影師奧賽羅 · 馬爾泰利曾經公然宣稱自己“厭惡色彩”，並稱色彩是“骯髒下流的玩意”。而在彩色片誕生之初，特藝色彩三棱鏡系統也使得彩色攝影機極爲龐大，而且色彩效果過於誇張，顯得並不真實。不過，在傑克 · 卡迪夫、戈登 · 威利斯的手中，色彩慢慢成爲了表情達意的手段和方法。

——技術上的問題，終究會被技術攻克；藝術上的事情，在藝術家手中，終究會變成藝術語言。

在經過幾十年的發展之後，聲音和色彩已經成爲了電影的組成部分。假若在這個時代，拍攝一部黑白電影或者是一部默片，那麼一定是導演的藝術理念在作祟，而不是技術手段的限制。由此來反觀 3D 這個技術性的命題，從 3D rig 到雙鏡頭攝影機，甚至是卡梅隆宣稱的，《阿凡達 2》要是用“裸眼 3D”技術——3D 正在經歷着自己的技術革新。

在未來，這個商業“元素”是不是會變成電影的“固定組成部分”還是一個未知數。

《阿凡達 2》

如果說，3D 技術的未來是樂觀的話，那麼“3D 藝術”的未來，則沒有那麼光明瞭。要成爲藝術手段，需要參與敘事，並且表情達意，並最終參與並影響影片的“表現力”。但是無論是從《變形金剛》還是從《皮娜》，甚至是戈達爾的《再見，語言》來看，3D 並沒有參與到影片的“藝術層面”之上。

那句“不看 3D 就等於沒看”，成了一句空洞的“廣告語”。無論是商業片導演的“被 3D”還是藝術片導演的“實驗試水”，3D 一直以來都是一個可有可無的“小玩意”。這個“小玩意”可以撬動市場和資本，可以令人不爽，可以讓票價上漲。唯一不能的，就是讓影片變的更加好看，讓人更加“身臨其境”。

從目前的情況來看，3D 漸漸遇冷，是一個不爭的事實。言及此，美國媒體表示稱，人們還是希望看到質量更高的平面圖像，而不是立體圖像。所以 4K 和 8K 方興未艾，3D 則遭遇了寒流。

美國媒體認爲，在未來，會有更多的導演迴歸到 2D 電影的範疇之中，但 3D 並不會消失，畢竟還有卡梅隆這樣執着於 3D 的導演。可以暢想的是，在未來，假若 3D 不再是製片廠強加的“任務”，而是導演“有意爲之”的訴求，那麼 3D 成爲電影語彙，並被寫進電影教科書，或許真的是“有生之年”的一個可以看到的事件。
