趁肯德基 0 元 bug 狂下單的人，別偷笑了，商家是可以不作數的

如何看待肯德基蛋撻 0 元 bug 後，瘋狂點 50、70 蛋撻的這種行爲？

單純從法律角度來說，商家可以主張“零元買蛋撻”的買賣合同關係因重大誤解而產生，因此可以撤銷；或者乾脆將這一法律關係定義爲“贈與合同”，進而主張可以撤銷。

我國《民法總則》第一百四十七條規定，“基於重大誤解實施的民事法律行爲，行爲人有權請求人民法院或者仲裁機構予以撤銷。” 而在具體的部門法方面，《合同法》第五十四條規定，“因重大誤解訂立的”合同經當事人向法院或者仲裁機構申請可以變更或撤銷。

爲什麼基於“重大誤解”的合同可以撤銷呢？這一規定涉及到法律上合同成立的基本要件：雙方就主要條款達成合意。在題目中，商家並沒有要白送蛋撻的意思表示，其內心的意思還是想要賣錢的，僅僅是因爲系統出現 bug，而錯誤地導致顧客可以零元下單，因此不能認爲達成了合意。

退一步說，即使認爲商家主觀上真是想要白送蛋撻，這類贈與合同在實際履行之前也可以撤銷。根據《合同法》第一百八十六條第一款，“贈與人在贈與財產的權利轉移之前可以撤銷贈與。” 也就是說，哪怕商家答應白白將商品送給消費者，在實際把蛋撻交到顧客手上之前，也可以完全合法地撤銷。

上述條款有一個例外：“具有救災、扶貧等社會公益、道德義務性質的贈與合同或者經過公證的贈與合同，不適用前款規定。” 然而，在題目中提到的情況下，顯然不符合這一例外。

 這是我國法律的情況，假設是在英美法系國家，可能被認爲合同根本就沒有成立。在英美法系中，買賣合同必須要有“對價”（Consideration）, 一方必須承擔了一定的給付義務，才能獲得合同上的權利。通俗來說，就是沒有付出就沒有收穫，不能空手套白狼。像這樣消費者一分錢不花就可以下單的情況，在英美法系國家可能被視爲根本沒有成立。（當然，這裏只是發散一下，我國法律沒有直接要求存在對價，但也會考慮公平的因素。例如，合同法規定“顯失公平”的合同可以撤銷。）

 當然，上面說的是純粹理論上的情況，在真實世界，再家大業大的企業，也未必真爲了幾個蛋撻的錢就起訴到法院要求撤銷。出於公衆形象的考慮，企業也犯不着和消費者較真，甚至可以將其作爲一場有正面意義的營銷事件。

然而，對方不一定較真，不代表消費者就可以心安理得地大薅羊毛。不給告上法庭，僅僅是一個人行爲準則的最低底線，而不是上限。子曾經曰過，“取乎其上，得乎其中；取乎其中，得乎其下；取乎其下，則無所得矣”。這個道理也無需多言了。
