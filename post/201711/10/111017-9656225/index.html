「喬布斯去世後蘋果都沒創新了，你看這個 iPhone X 的劉海，醜死了」

如何評價 iPhone X ？

近年來網絡上對蘋果產品的評價陷入了一種『找茬式』的評價方法。就拿去年的 MacBook 來說吧，聽說變薄了？索尼十年前的產品就比你薄了（雖然 vaio 已經倒閉了）。聽說邊框變窄了？我戴爾 XPS 三年前就這樣了（雖然 xps 爲了窄邊框把攝像頭放到了屏幕底部導致視頻通話只能拍到大鼻孔）。聽說蘋果做了個 TouchBar？我聯想兩年前就有看起來差不多的東西了哼。雖然一個是具有獨立芯片能獨立運行並且第一次把基於 ARM 和 Intel 的系統整合到一起，另外一個就是個 LED 燈條，但是兩個看起來不是差不多嘛，蘋果真是沒創新，這都要抄。

同樣的，這種『找茬式』的評價也在 iPhone X 發佈之後充斥在各種媒體上。你有全面屏？『抄小米 mix 的吧』。你有人臉識別？『安卓四年前玩剩下的了』。你有 OLED 屏幕？『求着三星人家才賣給你的吧』。無線充電？『我們都玩膩的東西你們還當寶貝呢』。更別提那個劉海了，『我雖然不買，但是看見你們這些果粉和個傻子一樣這麼醜的手機還排隊去搶，我都氣得跺腳』。

有很多不怎麼關注科技行業的人看多了那些評論可能就會想，『既然蘋果毫無創新，還總是撿別人剩下的，怎麼還沒倒閉呢』。不僅沒倒閉，股價還一路高歌猛進，距離成爲第一家市值萬億的公司近在咫尺。

有的時候你不得不感嘆科技屆的風向變幻之快。就拿蘋果最大的競爭對手三星來說吧，Galaxy S5 雖然實現了銷量突破，但在設計創新上都乏善可陳，被大家嘲笑是『塑料の魔術師』，口碑跌倒谷底。S6 雖然材質上有進步，但設計毫無亮點，反倒是雙曲屏版本的 S6 Edge 讓大家有點新鮮感。三星靈敏的市場嗅覺發現了這點，於是 S7 將雙曲面作爲主打機型，實現了口碑的逆轉。再加上 S8 雙曲面加全面屏的設計，三星終於徹底摘掉了這麼多年的『蘋果跟隨者』的名字。

而蘋果呢，雖然喬布斯去世之後，蘋果沒創新的論調就一直都有，但爆發點還是在 iPhone 7 發佈前後。蘋果打破了兩年一變設計的傳統，沒有拿出全新設計的 iPhone，再加上 新的 MacBook 也被詬病創新不足，而且還漲價了，蘋果一下子就成了毫無創新利慾薰心的奸商。

三星的口碑逆轉和蘋果的光環消失幾乎都是這短短兩年之間的事，那麼 iPhone X 能不能讓那些持有『蘋果無創新論』的人閉嘴呢？那麼就讓我們來說說 iPhone X 吧。在這之前，如果要我用一個詞來形容 iPhone X，那就是『激進』。是的，激進，在蘋果身上久違了的一個詞。

iPhone X notch · 劉海

是的，要說 iPhone X，第一個要說的就是這個侵入了屏幕的缺口。蘋果管它叫 sensor hub，說集成了好多高科技的感應器。普羅大衆給他起了個暱稱『劉海』，說它醜。

在我開始給這個劉海『洗地』之前，我想先說另一個東西，窄邊框。

在全面屏這個潮流之前，窄邊框可以說是衆多廠商的極致追求了。大家都要比拼側邊框的粗細，以及屏幕和邊框之前黑邊的大小。在這樣一種對『細』的追求之中，iPhone 巋然不動的粗大邊框從『好粗』變成了『邊框能跑馬』再到後來的『能停航母』。你不得不感嘆科技進步得如此之快。

但是仔細一想，窄邊框除了提升誤觸率之外，給用戶帶來了什麼真正的改變嗎？好像確實沒什麼吧。最大的用處大概就是拍真機照的時候顯得好看一點。但是你聽過 iPhone 用戶抱怨邊框粗嗎？很少吧。爲什麼？因爲用戶在使用手機的時候，注意力是會集中在屏幕顯示的內容上的。而不是一邊刷微博一邊仔細端詳這個屏幕邊框的粗細。

同樣的，對於 iPhone X 上的這個劉海，當用戶真正用起來的時候，他們也是不會去注意的。事實上幾乎所有的評測裏都提到，在使用一段時間之後，連他們自己都驚訝地發現自己幾乎不會在意這個劉海的存在。

所以沒買的 iPhone X 人會覺得，你們看不到那個劉海嗎，你們怎麼能忍啊！而對於買了 iPhone X 的人，看得到，但是注意不到。

有個心靈雞湯是這麼說的，白紙上一個墨點，有的人只看到這張紙被弄髒了，而有的人卻能看到墨點之外還是一片乾淨的白紙。這個劉海就是 iPhone X 全面屏上的那個『墨點』，你想怎麼看它呢？（爲了給這個劉海洗地我連心靈雞湯都用上了我真的很拼）

那麼這個劉海的設計有什麼優點呢。（是的我甚至覺得這個劉海設計是有優點的）。最大的一個優點，就是從『觀感』上實現了真正的全面屏。

如果蘋果爲了把屏幕做平，而保留上額頭會是什麼效果呢：

人眼對於視覺的感受是很微妙的。你們只看到了劉海，卻沒看到劉海兩邊的屏幕。這種設計讓屏幕延伸到了手機的最頂端，完全覆蓋了四個角落。這樣一來保證了上下設計在視覺感受上的對稱，而且會讓人覺得這個手機正面全部都是屏幕，只不過中間缺了一小塊。而保留上額頭的做法，則完全沒有全面屏的震撼感。

第二個優點，也就是 iPhone X 激進的體現。那就是屏佔比一步到位。

iPhone X 基本上把所有可能做成屏幕的地方都做成屏幕了。以至於我看慣了 iPhone X 之後，再看 Galaxy S8（個人認爲現在最好看的安卓旗艦），會覺得這不就是上下邊框窄了一點的普通手機麼。

今年出現了一大羣拿着 18：9 屏幕自稱全面屏的安卓手機，這也給正面和背面指紋之爭畫上了一個暫時的句號。原因嘛很簡單，正面放不下了啊……這也讓 Google 開心了一下，推了一萬年的虛擬鍵大家終於要用了。

但是如果思考安卓旗艦下一步進化的方向，很明顯這個下巴是多餘的。如果去掉下巴只保留額頭，整機的不對稱感會非常強烈，很不協調（說的就是小米 mix 夏普 essential phone 那一批）。所以我預測，在安卓旗艦開始去掉下巴的時候，上額頭很可能也會採用蘋果這種保留一塊缺口給攝像頭的設計。

第三點（我還能掰出第三點連我都佩服我自己），也是聽上去最牽強但也是最重要的一點，那就是爲了『品牌認知度』

在這之前，iPhone 的圓形 Home 鍵就是 iPhone 的標誌。幾乎沒有其他廠商敢去用相同的形狀。爲什麼？因爲 Apple 這個品牌太強勢了，如果你用了相同的設計，只會被當成抄襲者。那麼現在 Home 鍵消失之後，iPhone 新的標誌是什麼呢？蘋果給出的答案就是這個『劉海』。

其實你看看蘋果給出的宣傳材料就知道了，官網毫不避諱的給這個劉海全屏特寫。在官網以及 iOS 裏使用的 iPhone X 標誌，甚至放大了這個劉海的比例。包括現在媒體上對於這個劉海無數的討論，其實都是在幫助蘋果把這個設計與自身的品牌烙印在一起。是的，這個『劉海』就像是 iPod 時期的白色耳機，MacBook 的發光 LOGO 一樣，是未來 iPhone 的標誌了。

iPhone X · FACE ID

說完劉海，自然就要說 Face ID。我是一個接受新技術很快的人，所以我對於 Touch ID 的消失沒什麼留戀。

現在對於 Face ID 的指責很大一部分是來自於人們對於一個陌生的新事物的牴觸。比如他們會說,『刷臉就能解鎖，那你睡覺的時候別人不就能解鎖了嗎？Face ID 真垃圾還是指紋好』。你告訴他，Face ID 要求人睜眼看手機纔會解鎖，他們就會說『那你睡覺的時候，別人把手機放在你面前，然後叫醒你，你一睜眼不久手機被解鎖了嗎？Face ID 真垃圾還是指紋好』。通過不斷添加特定條件來創造出一個 Face ID 會失敗的場景，並不能證明這項技術不夠好。比方之前這個場景裏，解鎖睡着的人的手機用指紋不是更容易嗎？

在此我沒有興趣去對比二者的成功率或者解鎖速度，我更感興趣的是這兩種解鎖方式背後的交互邏輯。Face ID 比 Touch ID 更先進的原因並不是因爲它更安全或者解鎖速度更快，而是因爲 Face ID 是在『做減法』。

爲什麼指紋解鎖會流行起來？因爲按一下指紋比輸入 6 位的解鎖密碼更快。所以每個人都更喜歡指紋。但你有沒有想過，其實『把手指放到指紋上進行識別』也不是一個自然的動作。而 Face ID 則真正做到了在用戶無感知的情況下保護用戶的數據。

從 iOS 11 開始，鎖屏界面其實已經不再單是鎖屏界面了。它現在的角色應該被稱作通知中心更合適。蘋果希望你在點亮屏幕的一瞬間就能在這裏看到你收到的消息。而在這個過程中，指紋識別其實在交互邏輯上已經有很多衝突了。

最簡單的，如果你要打開一條通知，你需要①點亮屏幕②滑動通知打開③驗證指紋解鎖。雖然蘋果有通過『擡起喚醒』和『按下 home 解鎖』等方式來試圖簡化這個過程，但都不如 Face ID 來得徹底，點亮屏幕，打開通知。Face ID 讓很多操作都簡化了不少。

當然了，說了這麼多 Face ID 的好，也並不是說 Touch ID 有什麼不好。但是嚷嚷着沒有指紋不買的人，總讓我想起若干年前那些說沒有實體鍵盤不買的人。

iPhone X · 屏幕

這次蘋果的屏幕是自行設計，三星代工。於是我總是時不時地看到這樣的言論，『iPhone X 的屏幕肯定沒有三星好啊，不然三星還賣什麼』『三星肯定都是把最好的屏幕留給自己用的，怎麼會賣給蘋果』。

其實說這些話的人，應該是完全不瞭解產業鏈吧。三星電子（賣手機那個）和三星顯示（給三星蘋果生產顯示屏）分別是不同的子公司。三星顯示不會爲了另一家子公司犧牲自己最大客戶的利益。

說三星顯示爲了保護三星電子的利益只給蘋果二流產品，有點像說 OPPO 爲了保護 Vivo 的銷量自己只賣二流產品一樣。至於說會被三星掐着脖子限制供貨，估計這些人沒聽過『合同』這兩個字吧。其實『被三星掐脖子』這種說法本身就挺可笑的，明明是自己下訂單量太小優先級不夠，拿不到貨，反而覺得是別人要針對他一樣。

扯遠了，繼續說屏幕。那麼這個屏幕素質怎麼樣呢？專業評測機構 DisplayMate 給出非常高的評價：

According to DisplayMate, the iPhone X has the "most innovative and high performance" smartphone display it has ever tested. iPhone X matched or set new smartphone display records in the following categories: highest absolute color accuracy, highest full screen brightness for OLED smartphones, highest full screen contrast rating in ambient light, and highest contrast ratio. It also had the lowest screen reflectance and smallest brightness variation with a viewing angle. iPhone X 在以下類別裏創造或持平了最高紀錄：最高的絕對色彩準確度， OLED 屏幕裏最高的全屏亮度，最高的對比度，最低的屏幕反射率，在不同角度下最小的亮度差異。

下圖爲Note 8 和 IPhone X 色準曲線。可以看到 iPhone X 幾乎和標準曲線完全重疊。而 Note 8 則有微小的差異。當然了，這也有 iPhone X 色彩管理的功勞，iPhone X 會根據文件類型自動選擇 sRGB 或者 DCI-P3 模式，從而實現更準確的色彩。

有的人可能會覺得很難理解，爲什麼三星做了那麼多年 OLED 怎麼一上來就被蘋果超過了。其實也不難理解，三星給蘋果代工了那麼多年處理器，哪代獵戶座超過了蘋果 A 系列處理器的性能了嗎？

DisplayMate 評測總結是這麼說的：這塊屏幕從視覺效果上幾乎找不出任何缺點，可以用完美來形容。這塊屏幕很可能比你見過的任何手機屏幕、顯示器或高清電視都要好上一大截。

（寫得這麼不含蓄我都懷疑他們是不是收了錢，結果上他們官網一看，那種樸素的理工直男氣息撲面而來，我的疑慮立刻就消失了）

iPhone X · 交互

去掉 Home 鍵意味着在 iPhone 上使用了 10 年之久的交互邏輯也要改變了。取而代之的是這個細長的控制條，可以快速實現之前 home 鍵的各種功能。有的人覺得這讓 iPhone 的交互變得複雜了。其實並沒有，看下圖

可以看到，蘋果將原來 home 鍵的功能分解成了不同的手勢，使其適應了新的界面。好多人抱怨這學起來太麻煩了。我不禁感嘆，蘋果到底把用戶慣成了什麼樣子啊。這是 10 年來蘋果第一次改變 iPhone 上的交互方式。說真的，10 年學一次怎麼用新手機真的是不可想象的麻煩事兒嗎？你還記得你十年前第一次見到 iPhone 的興奮感嗎？你驚呼着這一個按鍵就能完全操控手機，樂此不疲地用雙指縮放剛拍的照片，那時候學習用新 iPhone 是一件新鮮而激動的事。那麼就不要覺得學習用 iPhone X 是一件麻煩事兒吧。

其實在看到 iPhone X 上這個控制條的第一眼，我立刻就想到了 iOS 在 iPad 上的多任務界面。在 iPad 上拖動多任務時，也會有這麼一個控制條出現在窗口的頂部。

這樣一來 iPhone X 的設計理念就很清晰了。每一個應用程序其實就是一張卡片，按住這個控制條就能移動這張卡片，左右滑動切換卡片，上滑收起卡片。整個操作是非常合乎邏輯的。

說到這裏我還有點私貨想說，明明 Android 虛擬鍵搞了這麼多年，結果呢，搞的是個啥？就是把三個實體鍵照搬到屏幕裏？除了佔屏幕空間有什麼其他用處嗎？怪不得安卓廠商都不領情呢。希望 Google 看到 iOS 之後明年也能做點交互上的創新吧。

iPhone X · Apple GPU 和內在的鉅變

很多人都沒注意到，甚至 Apple 自己也沒怎麼宣傳的地方，那就是這次 A11 的 GPU 是完全由蘋果自己製造的。不知道大家還記不記得今年（17 年）四月份的時候，Apple 芯片的 GPU 提供商 Imagination Technology 宣佈蘋果將在兩年內完全停止使用 Imagination 的芯片，然後股價暴跌大幅裁員然後被收購的 drama 嗎？結果，這才過了不到半年，蘋果就拿出來了完全由自己設計製造的 GPU，並且依然比去年的 A10 GPU 快了 30%。感受到 Apple 恐怖的研發能力了嗎？

在最近一次採訪中，Apple 負責硬件工程的副總裁 Dan Riccio 說到，iPhone X 的研發其實從 2014 年就開始了，而在去年（16 年）11 月左右便已經定型。而這段時間不正是蘋果被罵沒創新，被各路媒體唱衰得最起勁的時間嗎？默默攢了個大招卻又不能說，一定憋得很難受吧……

iPhone X 還能說的還有很多，迄今爲止最小的雙層 PCB 主板、在和 iPhone 8 相近的機身裏塞進了比 iPhone 8 Plus 還大的 L 形電池、自行設計的圖像傳感器能拍攝許多高端相機也無法拍攝的 4K@60 FPS 視頻、從雙色閃光燈到四色再到今年的 Slow Sync Flash、打破 DxOMark 記錄獲得了 101 分得拍照效果、第一個大規模生產的景深攝像頭，當然，怎麼能忘了刷爆社交媒體的 Animoji。

發佈會上每年都會說一遍的迄今爲最好的 iPhone ，今年顯得無比誠意。
